# Урок №144. Правило п'яти (Rule of Five, C++11)

У C++11 додали переміщення. Правило п'яти: якщо потрібен деструктор — реалізуйте (або заборонить) всі п'ять:

1. Деструктор
2. Конструктор копіювання
3. Оператор присвоювання (копіювання)
4. Конструктор переміщення
5. Оператор присвоювання (переміщення)

## Приклад: всі п'ять

```c++
class Buffer
{
    int* data;
    int size;
    
public:
    // Конструктор
    Buffer(int s) : data{new int[s]{}}, size{s} {}
    
    // Деструктор
    ~Buffer() { delete[] data; }
    
    // Копіювання
    Buffer(const Buffer& other)
        : data{new int[other.size]{}}
        , size{other.size}
    {
        for (int i = 0; i < size; ++i)
            data[i] = other.data[i];
    }
    
    Buffer& operator=(const Buffer& other)
    {
        if (this == &other) return *this;
        delete[] data;
        size = other.size;
        data = new int[size];
        for (int i = 0; i < size; ++i)
            data[i] = other.data[i];
        return *this;
    }
    
    // Переміщення
    Buffer(Buffer&& other) noexcept
        : data{other.data}
        , size{other.size}
    {
        other.data = nullptr;
        other.size = 0;
    }
    
    Buffer& operator=(Buffer&& other) noexcept
    {
        if (this == &other) return *this;
        delete[] data;
        data = other.data;
        size = other.size;
        other.data = nullptr;
        other.size = 0;
        return *this;
    }
};
```

## Copy-and-swap з переміщенням

```c++
class Buffer
{
    int* data;
    int size;
    
    void swap(Buffer& other) noexcept
    {
        std::swap(data, other.data);
        std::swap(size, other.size);
    }
    
public:
    Buffer(int s) : data{new int[s]{}}, size{s} {}
    
    ~Buffer() { delete[] data; }
    
    Buffer(const Buffer& other) : data{new int[other.size]{}}, size{other.size}
    {
        for (int i = 0; i < size; ++i) data[i] = other.data[i];
    }
    
    Buffer(Buffer&& other) noexcept : data{nullptr}, size{0}
    {
        swap(other);
    }
    
    Buffer& operator=(Buffer other)  // копія або переміщення
    {
        swap(other);
        return *this;
    }
};
```

## Використання переміщення

```c++
int main()
{
    Buffer a(100);       // конструктор
    Buffer b = a;        // копіювання
    Buffer c = std::move(a);  // переміщення (a стає порожнім)
    
    b = Buffer(200);     // тимчасовий → переміщення
    b = c;               // копіювання
    
    return 0;
}
```

## noexcept

Конструктори/оператори переміщення мають бути noexcept:

```c++
Buffer(Buffer&& other) noexcept;
Buffer& operator=(Buffer&& other) noexcept;
```

Це дозволяє стандартній бібліотеці ефективно переміщувати (наприклад, `std::vector`).

## Заборона переміщення

```c++
class NotMovable
{
public:
    NotMovable(NotMovable&&) = delete;
    NotMovable& operator=(NotMovable&&) = delete;
};
```

## Правило нуля

Якщо клас не керує ресурсами — не пишіть нічого (Rule of Zero):

```c++
class Person  // Rule of Zero
{
    std::string name;
    int age;
    std::vector<int> scores;
};
```

## Правило

Якщо клас керує ресурсами — реалізуйте всі п'ять функцій. Якщо ні — не реалізуйте жодної (Rule of Zero).
