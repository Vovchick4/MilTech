# Урок №121. Класи, Об'єкти і Методи

## Клас

`Клас` — це тип даних, який об'єднує дані (поля) та функції (методи).

```c++
class Circle
{
    double radius;          // поле (дані)
    
public:
    void setRadius(double r) { radius = r; }  // метод
    double getArea() { return 3.14159 * radius * radius; }
};
```

## Об'єкт

`Об'єкт` — конкретний екземпляр класу:

```c++
Circle c1;          // об'єкт
Circle c2;          // інший об'єкт

c1.setRadius(5.0);
c2.setRadius(10.0);

std::cout << c1.getArea();  // 78.53975
std::cout << c2.getArea();  // 314.159
```

## Методи

Методи — це функції, визначені всередині класу:

```c++
class Rectangle
{
    double width, height;
    
public:
    void setDimensions(double w, double h)
    {
        width = w;
        height = h;
    }
    
    double getArea() { return width * height; }
    double getPerimeter() { return 2 * (width + height); }
};
```

## Визначення методів зовні класу

```c++
class Rectangle
{
    double width, height;
public:
    void setDimensions(double w, double h);
    double getArea();
};

// Визначення методів:
void Rectangle::setDimensions(double w, double h)
{
    width = w;
    height = h;
}

double Rectangle::getArea()
{
    return width * height;
}
```

## Ключове слово this

`this` — вказівник на поточний об'єкт:

```c++
class Example
{
    int value;
public:
    void setValue(int value)
    {
        this->value = value;  // this->value — поле, value — параметр
    }
};
```

## Створення об'єктів

```c++
// На стеку:
Circle c1;
Circle c2(5.0);  // з конструктором

// На купі:
Circle* c3 = new Circle();
delete c3;

// Через unique_ptr:
auto c4 = std::make_unique<Circle>();
```

## Правило

Клас має представляти одну чітку сутність (Single Responsibility). Методи мають працювати з даними цього класу.
