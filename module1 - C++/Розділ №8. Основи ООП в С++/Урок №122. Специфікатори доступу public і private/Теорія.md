# Урок №122. Специфікатори доступу public і private

Специфікатори доступу контролюють, хто може звертатися до членів класу.

## Три рівні доступу

| Специфікатор | Доступ |
|-------------|--------|
| `public` | Будь-хто |
| `private` | Тільки методи цього класу |
| `protected` | Цей клас і похідні (наслідування) |

## public

Доступні звідусіль:

```c++
class PublicExample
{
public:
    int x;
    void print() { std::cout << x; }
};

PublicExample obj;
obj.x = 5;       // OK
obj.print();     // OK
```

## private

Доступні тільки всередині класу:

```c++
class PrivateExample
{
private:
    int secret;  // тільки методи класу
    
public:
    void setSecret(int s) { secret = s; }
    int getSecret() const { return secret; }
};

PrivateExample obj;
// obj.secret = 5;  // ПОМИЛКА! private
obj.setSecret(5);     // OK через метод
```

## struct vs class

```c++
struct Point { int x; int y; };   // public за замовчуванням
class  Point { int x; int y; };   // private за замовчуванням!
```

## За замовчуванням

```c++
class MyClass {
    int x;  // private за замовчуванням
public:
    int y;  // явно public
};
```

## Принцип мінімальних привілеїв

```c++
class BankAccount
{
private:
    double balance;  // приховано — не можна змінити напряму
    
public:
    void deposit(double amount)  { if (amount > 0) balance += amount; }
    bool withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            return true;
        }
        return false;
    }
    double getBalance() const { return balance; }
};
```

## Getters & Setters

```c++
class Student
{
    std::string name;
    int age;
    
public:
    void setName(const std::string& n) { name = n; }
    std::string getName() const { return name; }
    
    void setAge(int a) { if (a > 0 && a < 150) age = a; }
    int getAge() const { return age; }
};
```

## Правило

Робіть поля `private` за замовчуванням. Надавайте доступ тільки там, де це дійсно потрібно.
