# Урок №129. Прихований вказівник this

`this` — це прихований вказівник, який передається в кожен нестатичний метод і вказує на поточний об'єкт.

## Як це працює

```c++
class Point
{
    int x, y;
    
public:
    void setX(int x)
    {
        // this → вказує на об'єкт, для якого викликано метод
        this->x = x;  // розрізняє поле і параметр
    }
    
    void setY(int y) { this->y = y; }
};
```

## Використання this

### 1. Розрізнення імен

```c++
class Example
{
    int value;
    
public:
    void setValue(int value)
    {
        this->value = value;  // поле = параметр
    }
};
```

### 2. Ланцюжковий виклик методів (method chaining)

```c++
class Calculator
{
    int result = 0;
    
public:
    Calculator& add(int x)  { result += x; return *this; }
    Calculator& sub(int x)  { result -= x; return *this; }
    Calculator& mul(int x)  { result *= x; return *this; }
    
    int getResult() const { return result; }
};

int main()
{
    Calculator calc;
    int r = calc.add(5).mul(3).sub(2).getResult();  // (5*3)-2 = 13
    return 0;
}
```

### 3. Передача *this в іншу функцію

```c++
class Node
{
    int value;
    Node* next = nullptr;
    
public:
    Node(int v) : value{v} {}
    
    void addToList(List& list)
    {
        list.add(this);  // передаємо поточний об'єкт
    }
};
```

## this — константний вказівник

```c++
class Demo
{
public:
    void method()
    {
        // this = nullptr;  // ПОМИЛКА! this — константний вказівник
        // this = &other;   // ПОМИЛКА!
    }
    
    void constMethod() const
    {
        // У const-методах this — const Demo* const
        // this->field = 5;  // ПОМИЛКА!
    }
};
```

class Movable

## Приклад: оператор присвоювання

```c++
class MyClass
{
    int* data;
    
public:
    MyClass& operator=(const MyClass& other)
    {
        if (this != &other)  // перевірка на самоприсвоювання
        {
            delete[] data;
            data = new int[*other.data];
            // копіювання...
        }
        return *this;
    }
};
```

## Правило

`this` дозволяє методу знати, для якого об'єкта він викликаний. Використовуйте `this->` для розрізнення імен і `*this` для повернення поточного об'єкта.
