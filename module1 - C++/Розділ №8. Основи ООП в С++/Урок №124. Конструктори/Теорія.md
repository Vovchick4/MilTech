# Урок №124. Конструктори

`Конструктор` — спеціальний метод, який викликається при створенні об'єкта.

## Синтаксис

```c++
class Point
{
    int x, y;
    
public:
    Point()  // конструктор без параметрів (за замовчуванням)
    {
        x = 0;
        y = 0;
    }
};
```

## Конструктор з параметрами

```c++
class Point
{
    int x, y;
    
public:
    Point(int xVal, int yVal)
    {
        x = xVal;
        y = yVal;
    }
};

int main()
{
    Point p1;           // виклик Point()
    Point p2(5, 10);    // виклик Point(5, 10)
    Point p3 {5, 10};   // uniform-ініціалізація
    return 0;
}
```

## Список ініціалізації (кращий спосіб)

```c++
class Point
{
    int x, y;
    
public:
    Point(int xVal, int yVal) : x{xVal}, y{yVal}  // список ініціалізації
    {
        // тіло конструктора порожнє
    }
};
```

## Конструктор за замовчуванням

Якщо не визначено жодного конструктора — компілятор створює неявний:

```c++
class A { int x; };  // неявний конструктор A() {}

// Якщо визначено хоч один — неявний не створюється:
class B {
    int x;
public:
    B(int val) : x{val} {}
};

// B b;   // ПОМИЛКА! немає конструктора за замовчуванням
B b(5);  // OK
```

## Кілька конструкторів (перевантаження)

```c++
class Rectangle
{
    double width, height;
    
public:
    Rectangle() : width{0}, height{0} {}
    Rectangle(double side) : width{side}, height{side} {}  // квадрат
    Rectangle(double w, double h) : width{w}, height{h} {}
};

int main()
{
    Rectangle r1;           // 0x0
    Rectangle r2(5.0);      // 5x5 (квадрат)
    Rectangle r3(4.0, 3.0); // 4x3
    return 0;
}
```

## Конструктор копіювання (простий)

```c++
class Point
{
    int x, y;
public:
    Point(int x, int y) : x{x}, y{y} {}
    
    // Конструктор копіювання (компілятор створює неявно)
    Point(const Point& other) : x{other.x}, y{other.y} {}
};

Point p1(5, 10);
Point p2(p1);  // копіювання
```

## Правило

Завжди ініціалізуйте всі поля в конструкторі (краще через список ініціалізації).
