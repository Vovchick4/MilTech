# Урок №131. Класи і const

## const методи

Метод, позначений `const`, гарантує, що не змінює об'єкт:

```c++
class Point
{
    int x, y;
    
public:
    int getX() const { return x; }  // const метод
    int getY() const { return y; }
    
    void setX(int val) { x = val; }  // не const
};
```

## Виклик const і не-const методів

```c++
Point p1;
p1.setX(5);     // OK
p1.getX();      // OK

const Point p2;
// p2.setX(5);  // ПОМИЛКА! const об'єкт → тільки const методи
p2.getX();      // OK
```

## const посилання на об'єкт

```c++
void printPoint(const Point& p)  // через const &
{
    std::cout << p.getX() << ", " << p.getY();  // OK (const методи)
    // p.setX(10);  // ПОМИЛКА! setX — не const
}
```

## mutable (зміна в const методі)

```c++
class Cache
{
    mutable int cacheValue = 0;  // можна змінювати навіть у const
    mutable bool cacheValid = false;
    
    int expensiveData;
    
public:
    int getData() const
    {
        if (!cacheValid)
        {
            cacheValue = expensiveData;  // OK: mutable
            cacheValid = true;
        }
        return cacheValue;
    }
};
```

## const об'єкти

```c++
class Circle
{
    double radius;
    
public:
    Circle(double r) : radius{r} {}
    double getArea() const { return 3.14159 * radius * radius; }
};

const Circle unitCircle(1.0);  // константний об'єкт
// unitCircle = Circle(2.0);  // ПОМИЛКА!
```

## const *this

У const методі `this` стає `const ClassName* const`:

```c++
class Demo
{
    int x;
    
public:
    void method() const
    {
        // x = 5;    // ПОМИЛКА!
        // this->x = 5;  // ПОМИЛКА!
    }
};
```

## Наслідування const

const методи можуть викликати тільки інші const методи:

```c++
class Logger
{
    void log() const { /* логування */ }
    
public:
    void process() const
    {
        log();  // OK (const → const)
    }
    
    void update()
    {
        log();  // OK (не const може викликати const)
    }
};
```

## Правило

Позначайте методи `const`, якщо вони не змінюють об'єкт. Це дозволяє використовувати їх з const об'єктами та посиланнями.
