# Урок №136. Вкладені типи даних в класах

В C++ в одному класі можна визначити інший клас або enum (вкладені типи).

## Вкладений клас

```c++
class Outer
{
public:
    class Inner  // вкладений клас
    {
    public:
        void print() { std::cout << "Inner\n"; }
    };
};

int main()
{
    Outer::Inner obj;  // створення об'єкта вкладеного класу
    obj.print();
    return 0;
}
```

## Вкладений enum

```c++
class Message
{
public:
    enum class Type  // вкладений enum
    {
        INFO,
        WARNING,
        ERROR
    };
    
private:
    Type type;
    std::string text;
    
public:
    Message(Type t, const std::string& msg) : type{t}, text{msg} {}
    
    Type getType() const { return type; }
};

int main()
{
    Message msg(Message::Type::INFO, "Все добре");
    return 0;
}
```

## Вкладений клас і доступ

Вкладений клас має доступ до членів зовнішнього (в C++11+):

```c++
class Tree
{
    struct Node  // вкладений, private — прихований
    {
        int data;
        Node* left = nullptr;
        Node* right = nullptr;
        
        Node(int d) : data{d} {}
    };
    
    Node* root = nullptr;
    
public:
    void insert(int value) { /* працює з Node */ }
};
```

## Приклад: ітератор

```c++
class IntVector
{
    int* data;
    int size;
    
public:
    class Iterator
    {
        int* ptr;
    public:
        Iterator(int* p) : ptr{p} {}
        int& operator*() { return *ptr; }
        Iterator& operator++() { ++ptr; return *this; }
        bool operator!=(const Iterator& other) { return ptr != other.ptr; }
    };
    
    Iterator begin() { return Iterator(data); }
    Iterator end()   { return Iterator(data + size); }
};

int main()
{
    IntVector vec;
    for (auto it = vec.begin(); it != vec.end(); ++it)
        std::cout << *it << " ";
    return 0;
}
```

## Принцип інкапсуляції

Вкладені типи допомагають приховати деталі реалізації:

```c++
class Database
{
    struct Connection  // прихований тип
    {
        std::string host;
        int port;
        bool connected = false;
    };
    
    Connection conn;
    
public:
    void connect(const std::string& host, int port)
    {
        conn.host = host;
        conn.port = port;
        // ...
    }
};
```

## Правило

Використовуйте вкладені типи для внутрішніх допоміжних структур та ітераторів, які тісно пов'язані із зовнішнім класом.
