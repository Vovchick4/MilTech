# Урок №133. Статичні методи класу

Статичні методи належать класу, а не об'єкту. Викликаються без створення екземпляра.

## Синтаксис

```c++
class Math
{
public:
    static double square(double x) { return x * x; }
    static double cube(double x) { return x * x * x; }
};

// Виклик без об'єкта:
double s = Math::square(5.0);  // 25
double c = Math::cube(3.0);    // 27
```

## Особливості

1. **Немає `this`** — не можуть звертатися до нестатичних членів
2. **Не можуть бути `const`** (немає об'єкта для const)
3. **Можна викликати без об'єкта**
4. **Можуть звертатися до статичних членів**

```c++
class Example
{
    int x = 5;
    static int s;
    
public:
    static void func()
    {
        // x = 10;  // ПОМИЛКА! немає this
        s = 10;      // OK: статичний член
    }
};
int Example::s = 0;
```

## Приклад: фабричний метод

```c++
class Logger
{
    std::string name;
    
    Logger(const std::string& n) : name{n} {}  // private конструктор
    
public:
    static Logger createConsole() { return Logger{"Console"}; }
    static Logger createFile()    { return Logger{"File"}; }
};

int main()
{
    auto log1 = Logger::createConsole();
    auto log2 = Logger::createFile();
    // Logger direct("Test");  // ПОМИЛКА! конструктор private
    return 0;
}
```

## Приклад: рандом

```c++
class Random
{
    static std::mt19937 gen;
    
public:
    static int get(int min, int max)
    {
        std::uniform_int_distribution<int> dist(min, max);
        return dist(gen);
    }
};

std::mt19937 Random::gen{std::random_device{}()};

int main()
{
    std::cout << Random::get(1, 10);  // випадкове 1-10
    return 0;
}
```

## Статичний поліморфізм (CRTP)

```c++
template<typename Derived>
class Base
{
public:
    static void identify()
    {
        std::cout << "Це " << Derived::name() << "\n";
    }
};

class Dog : public Base<Dog>
{
public:
    static std::string name() { return "Собака"; }
};
```

## Коли використовувати?

- **Фабричні методи** — створення об'єктів
- **Утиліти** — функції, що не потребують стану
- **Синглтон** — глобальна точка доступу
- **Лічильники** — робота зі статичними полями

## Правило

Статичні методи — це функції, логічно пов'язані з класом, але які не потребують екземпляра.
