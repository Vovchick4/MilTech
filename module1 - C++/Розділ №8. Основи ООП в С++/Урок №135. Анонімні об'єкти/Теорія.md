# Урок №135. Анонімні об'єкти

Анонімні (тимчасові) об'єкти — це об'єкти без імені, які існують тимчасово.

## Створення

```c++
int sum = 5 + 3;  // 5 і 3 — тимчасові об'єкти (int)
```

Власні анонімні об'єкти:

```c++
class Point
{
    int x, y;
public:
    Point(int x, int y) : x{x}, y{y} {}
    void print() const { std::cout << x << ", " << y; }
};

// Анонімний об'єкт:
Point{5, 10}.print();  // створюється, використовується, знищується
```

## Користь: зручність

```c++
class Calculator
{
public:
    int add(int a, int b) { return a + b; }
};

// З іменем:
Calculator calc;
int r1 = calc.add(5, 3);

// Анонімно:
int r2 = Calculator{}.add(5, 3);  // не зберігаємо калькулятор
```

## Повернення з функції

```c++
class Point
{
    int x, y;
public:
    Point(int x, int y) : x{x}, y{y} {}
    void print() const { std::cout << x << ", " << y; }
};

Point createPoint(int x, int y)
{
    return Point{x, y};  // анонімний об'єкт (RVO)
}

Point createPoint2(int x, int y)
{
    return {x, y};  // скорочена форма
}
```

## Передача в функцію

```c++
void drawPoint(const Point& p) { p.print(); }

int main()
{
    drawPoint(Point{10, 20});  // анонімний об'єкт
    drawPoint({30, 40});       // скорочена форма (C++11)
    return 0;
}
```

## У масивах і контейнерах

```c++
#include <vector>

struct Point { int x, y; };

int main()
{
    std::vector<Point> points;
    
    // Додавання анонімного об'єкта
    points.push_back(Point{5, 10});
    points.push_back({15, 20});  // скорочено
    
    // Ініціалізація списком
    std::vector<Point> pts = {{1, 2}, {3, 4}, {5, 6}};
    
    return 0;
}
```

## Час життя

Анонімні об'єкти живуть до кінця виразу:

```c++
struct Demo
{
    Demo() { std::cout << "Створено\n"; }
    ~Demo() { std::cout << "Знищено\n"; }
    void work() const { std::cout << "Робота\n"; }
};

int main()
{
    Demo{}.work();  // Створено → Робота → Знищено
    std::cout << "Після\n";  // Після
    return 0;
}
```

## const посилання подовжує життя

```c++
struct Demo { int x; };

int main()
{
    const auto& ref = Demo{5};  // життя подовжено до кінця області видимості
    std::cout << ref.x;  // OK
    return 0;
}
```

## Правило

Анонімні об'єкти зручні для одноразових операцій і короткого часу життя.
