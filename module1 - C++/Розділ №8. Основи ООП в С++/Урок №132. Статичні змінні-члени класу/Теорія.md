# Урок №132. Статичні змінні-члени класу

Статичні члени належать ВСЬОМУ класу, а не окремим об'єктам.

## Статичне поле

```c++
class Counter
{
    static int totalCount;  // оголошення (одне на всі об'єкти)
    
public:
    Counter() { ++totalCount; }
    ~Counter() { --totalCount; }
    
    static int getCount() { return totalCount; }
};

// Визначення (обов'язково! в .cpp файлі):
int Counter::totalCount = 0;
```

## Використання

```c++
int main()
{
    std::cout << Counter::getCount();  // 0
    
    Counter c1;
    Counter c2;
    
    std::cout << Counter::getCount();  // 2
    {
        Counter c3;
        std::cout << Counter::getCount();  // 3
    }
    std::cout << Counter::getCount();  // 2
    
    return 0;
}
```

## Статичне поле з in-class ініціалізацією

```c++
class Config
{
public:
    static const int MAX_SIZE = 100;        // const static — можна в класі
    static constexpr double PI = 3.14159;   // constexpr — можна в класі
};

// Для НЕ-const static ПОТРІБНЕ визначення зовні:
class Example
{
    static int counter;  // тільки оголошення
};
int Example::counter = 0;  // визначення в .cpp
```

## Доступ до статичних членів

```c++
class Math
{
public:
    static constexpr double PI = 3.14159;
    
    static double square(double x) { return x * x; }
};

// Доступ:
Math::PI;
Math::square(5.0);

// Через об'єкт (не рекомендується):
Math m;
m.PI;
```

## Статичні та нестатичні методи

```c++
class Demo
{
    int x = 5;
    static int s_x;
    
public:
    void nonStatic()
    {
        x = 10;      // OK: нестатичний метод має this
        s_x = 20;    // OK: доступ до статичного
    }
    
    static void staticMethod()
    {
        // x = 10;   // ПОМИЛКА! немає this
        s_x = 20;    // OK
    }
};
int Demo::s_x = 0;
```

## Приклад: ID генератор

```c++
class IDGenerator
{
    static int nextID;
    
public:
    static int generate() { return nextID++; }
};

int IDGenerator::nextID = 1;

int main()
{
    std::cout << IDGenerator::generate();  // 1
    std::cout << IDGenerator::generate();  // 2
    std::cout << IDGenerator::generate();  // 3
    return 0;
}
```

## Правило

Статичні члени — для даних і функцій, спільних для всіх об'єктів класу (лічильники, константи, фабрики).
