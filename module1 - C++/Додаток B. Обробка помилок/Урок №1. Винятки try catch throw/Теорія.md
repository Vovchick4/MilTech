# Урок №1. Винятки try catch throw

Винятки — механізм обробки помилок в C++. Дозволяє передати помилку вгору по стеку викликів.

## Базовий синтаксис

```c++
#include <iostream>
#include <stdexcept>

double divide(double a, double b)
{
    if (b == 0.0)
        throw std::runtime_error("Ділення на нуль!");
    return a / b;
}

int main()
{
    try
    {
        double result = divide(10, 0);
        std::cout << result;
    }
    catch (const std::exception& e)
    {
        std::cout << "Помилка: " << e.what() << "\n";
    }
    
    return 0;
}
```

## Типи винятків

```c++
try
{
    // код, що може викинути виняток
}
catch (int errCode)
{
    // обробка int
}
catch (const std::string& msg)
{
    // обробка string
}
catch (const std::exception& e)
{
    // обробка стандартних винятків
}
catch (...)
{
    // обробка ВСІХ винятків (ellipsis)
}
```

## throw — викидання

```c++
void checkAge(int age)
{
    if (age < 0)
        throw std::invalid_argument("Вік не може бути від'ємним");
    if (age > 150)
        throw std::out_of_range("Вік занадто великий");
    if (age < 18)
        throw std::runtime_error("Доступ заборонено: вік < 18");
}
```

## Розкручування стеку (stack unwinding)

```c++
void funcC() { throw std::runtime_error("Error in C"); }
void funcB() { funcC(); }  // виняток проходить крізь
void funcA() { funcB(); }  // виняток проходить крізь

int main()
{
    try { funcA(); }
    catch (const std::exception& e)
    {
        std::cout << e.what();  // "Error in C"
    }
    return 0;
}
```

## RAII і винятки

```c++
class FileHandler
{
    FILE* file;
public:
    FileHandler(const char* name)
    {
        file = fopen(name, "r");
        if (!file) throw std::runtime_error("Не вдалося відкрити файл");
    }
    
    ~FileHandler()
    {
        if (file) fclose(file);  // завжди викличеться (навіть при винятку)
    }
};
```

## noexcept

```c++
void safeFunction() noexcept  // гарантує: не викидає винятків
{
    // код без винятків
}

// Умовний noexcept
void func() noexcept(std::is_nothrow_swappable_v<T>);
```

## rethrow

```c++
try
{
    // код
}
catch (const std::exception& e)
{
    std::cout << "Логування: " << e.what() << "\n";
    throw;  // передати виняток далі
}
```

## Правило

Використовуйте винятки для виняткових ситуацій, не для контролю потоку.
