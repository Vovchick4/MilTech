# Урок №2. Класи винятків

Стандартна ієрархія винятків і створення власних.

## Стандартна ієрархія

```
std::exception
├── std::logic_error
│   ├── std::invalid_argument
│   ├── std::domain_error
│   ├── std::length_error
│   └── std::out_of_range
├── std::runtime_error
│   ├── std::range_error
│   ├── std::overflow_error
│   └── std::underflow_error
└── std::bad_alloc
```

## Власний клас винятку

```c++
#include <stdexcept>
#include <string>

class DivisionByZero : public std::runtime_error
{
public:
    DivisionByZero()
        : std::runtime_error("Ділення на нуль")
    {}
    
    DivisionByZero(const std::string& detail)
        : std::runtime_error("Ділення на нуль: " + detail)
    {}
};

double divide(double a, double b)
{
    if (b == 0)
        throw DivisionByZero("double");
    return a / b;
}
```

## Приклад: бібліотечний виняток

```c++
class MatrixError : public std::runtime_error
{
public:
    using std::runtime_error::runtime_error;
};

class MatrixDimensionError : public MatrixError
{
public:
    MatrixDimensionError(int r1, int c1, int r2, int c2)
        : MatrixError("Невірні розміри матриці: (" +
              std::to_string(r1) + "x" + std::to_string(c1) + ") + (" +
              std::to_string(r2) + "x" + std::to_string(c2) + ")")
    {}
};

void multiply(int r1, int c1, int r2, int c2)
{
    if (c1 != r2)
        throw MatrixDimensionError(r1, c1, r2, c2);
}
```

## Виняток з даними

```c++
class HttpError : public std::runtime_error
{
    int statusCode;
    
public:
    HttpError(int code, const std::string& msg)
        : std::runtime_error(msg)
        , statusCode{code}
    {}
    
    int getStatusCode() const { return statusCode; }
};

void handleRequest()
{
    try
    {
        // обробка запиту
        throw HttpError(404, "Not Found");
    }
    catch (const HttpError& e)
    {
        std::cout << "HTTP " << e.getStatusCode() << ": " << e.what();
    }
}
```

## Перехоплення по ієрархії

```c++
// Порядок має значення: від конкретного до загального
try
{
    // код
}
catch (const std::invalid_argument& e) { /* конкретно */ }
catch (const std::runtime_error& e)    { /* ширше */ }
catch (const std::exception& e)        { /* всі std */ }
catch (...)                            { /* все інше */ }
```

## Використання

```c++
class Database
{
public:
    void connect(const std::string& host)
    {
        if (host.empty())
            throw std::invalid_argument("Host не може бути порожнім");
        
        if (!tryConnect(host))
            throw std::runtime_error("Не вдалося під'єднатися до " + host);
    }
};

int main()
{
    Database db;
    try
    {
        db.connect("");
    }
    catch (const std::invalid_argument& e)
    {
        std::cout << "Помилка параметра: " << e.what() << "\n";
    }
    catch (const std::runtime_error& e)
    {
        std::cout << "Помилка виконання: " << e.what() << "\n";
    }
    
    return 0;
}
```
