# Урок №32. Тип даних void

`void` (з англ. "порожнеча") — це спеціальний тип у C++, який означає **відсутність типу**. Він не є звичайним типом даних, оскільки не можна створити змінну типу `void` для зберігання даних.

## Використання void

### 1. Функції, що не повертають значення

Найпоширеніше використання `void` — вказати, що функція не повертає жодного значення:

```c++
void printMessage()
{
    std::cout << "Hello, world!\n";
    // return не обов'язковий
}

int main()
{
    printMessage();  // виклик функції
    return 0;        // main() повертає int, не void
}
```

Функція `void` може використовувати `return;` для дострокового виходу:

```c++
void checkAge(int age)
{
    if (age < 18)
    {
        std::cout << "Доступ заборонено\n";
        return;  // достроковий вихід
    }
    std::cout << "Доступ дозволено\n";
}
```

### 2. Параметри void

Специфікація `void` у списку параметрів функції вказує, що функція не приймає жодних аргументів:

```c++
int getValue(void)  // стиль C, явно вказує "без параметрів"
{
    return 42;
}

int getValue()      // еквівалентно в C++
{
    return 42;
}
```

### 3. Вказівники void* (про безпечне використання — пізніше)

`void*` — це вказівник на дані без типу. Він може вказувати на будь-який тип даних, але його не можна розіменовувати без явного приведення:

```c++
int x = 5;
void* ptr = &x;     // void* може зберігати адресу будь-якого типу
// std::cout << *ptr;  // ПОМИЛКА! не можна розіменувати void*
int* intPtr = static_cast<int*>(ptr);  // потрібне явне приведення
```

## Чого НЕ МОЖНА робити з void

```c++
void value;         // ПОМИЛКА! не можна оголосити змінну типу void
void func();        // OK — прототип функції
void result = func();  // ПОМИЛКА! void не можна присвоїти змінній
void& ref;          // ПОМИЛКА! не можна мати посилання на void
```

## Таблиця використання void

| Використання | Опис |
|-------------|------|
| `void func()` | Функція нічого не повертає |
| `int func(void)` | Функція без параметрів (C-style) |
| `void* ptr` | Вказівник на дані без типу |
| `void` | Заборонено для змінних |

## Правило

Використовуйте `void` для позначення:
- Функцій, які виконують дію без повернення результату
- Функцій без параметрів (хоча в C++ дужки всередині — достатньо)
