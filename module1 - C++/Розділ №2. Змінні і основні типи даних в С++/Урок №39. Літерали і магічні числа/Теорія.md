# Урок №39. Літерали і магічні числа

## Літерали

`Літерал` — це фіксоване значення, записане безпосередньо у вихідному коді.

### Цілочисельні літерали

```c++
int a = 42;         // десятковий
int b = 052;        // вісімковий (починається з 0)
int c = 0x2A;       // шістнадцятковий (0x)
int d = 0b101010;   // двійковий (C++14+, 0b)
```

### Літерали з плаваючою крапкою

```c++
double a = 3.14;    // стандартний double
float b = 3.14f;    // f/F суфікс для float
double c = 3.14;    // за замовчуванням double
long double d = 3.14L;  // L суфікс для long double
```

### Символьні літерали

```c++
char c = 'A';       // символ
char nl = '\n';     // escape-послідовність
```

### Рядкові літерали

```c++
const char* str = "Hello";  // рядок C-style
std::string s = "Hello";    // рядок C++ (std::string)
```

### Логічні літерали

```c++
bool t = true;
bool f = false;
```

### nullptr літерал

```c++
int* ptr = nullptr;  // нульовий вказівник (C++11)
```

## Роздільники цифр (C++14)

Для кращої читабельності великих чисел:

```c++
int million = 1'000'000;      // 1 000 000
int hex = 0xFF'FF'FF'FF;      // шістнадцятковий
double pi = 3.141'592'653'59; // з плаваючою крапкою
```

## Магічні числа

`Магічне число` — це числовий літерал, який використовується без пояснення:

```c++
// Погано: магічні числа
double area = 3.14159 * r * r;
int maxStudents = 30;

// Добре: константи з іменами
const double PI = 3.14159;
const int MAX_STUDENTS = 30;
double area = PI * r * r;
int maxStudents = MAX_STUDENTS;
```

## Чому магічні числа — це зло?

1. **Незрозумілий код** — що означає число 86400?
2. **Складність змін** — потрібно змінювати в десятках місць
3. **Помилки** — легко пропустити одне число при зміні

```c++
// Погано: магічні числа
int totalSeconds = days * 86400;   // що таке 86400?
int hours = totalSeconds / 3600;   // а 3600?

// Добре: іменовані константи
const int SECONDS_PER_DAY = 86400;
const int SECONDS_PER_HOUR = 3600;
int totalSeconds = days * SECONDS_PER_DAY;
int hours = totalSeconds / SECONDS_PER_HOUR;
```

## Обмежені літерали

```c++
// Магічне число 5 зустрічається в коді
int scores[5];             // що означає 5?
for (int i = 0; i < 5; ++i)  // знову 5?

// Правильно
const int NUM_STUDENTS = 5;
int scores[NUM_STUDENTS];
for (int i = 0; i < NUM_STUDENTS; ++i)
```

## Правило

Ніколи не використовуйте "голі" числа в коді. Використовуйте константи зі змістовними іменами.
