# Урок №33. Розмір типів даних

Кожен тип даних у C++ займає певний об'єм пам'яті. Розмір типу залежить від компілятора, архітектури процесора та операційної системи.

## Оператор sizeof

`sizeof` — це оператор, який повертає розмір типу або змінної в байтах:

```c++
#include <iostream>

int main()
{
    std::cout << "Розміри типів даних (в байтах):\n";
    std::cout << "bool:      " << sizeof(bool) << "\n";
    std::cout << "char:      " << sizeof(char) << "\n";
    std::cout << "short:     " << sizeof(short) << "\n";
    std::cout << "int:       " << sizeof(int) << "\n";
    std::cout << "long:      " << sizeof(long) << "\n";
    std::cout << "long long: " << sizeof(long long) << "\n";
    std::cout << "float:     " << sizeof(float) << "\n";
    std::cout << "double:    " << sizeof(double) << "\n";
    return 0;
}
```

Типовий вивід на 64-бітній системі:
```
bool:      1
char:      1
short:     2
int:       4
long:      4 (Windows) / 8 (Linux/macOS)
long long: 8
float:     4
double:    8
```

## sizeof зі змінними

```c++
int x;
std::cout << sizeof(x);   // 4
std::cout << sizeof x;    // також працює (дужки необов'язкові для змінних)
```

## Байти, біти та діапазон значень

1 байт = 8 бітів. Кількість можливих значень = 2^(кількість_бітів).

| Тип | Розмір | Бітів | Мінімум | Максимум |
|-----|--------|-------|---------|----------|
| `bool` | 1 байт | 8 | false (0) | true (1) |
| `char` | 1 байт | 8 | -128 або 0 | 127 або 255 |
| `short` | 2 байти | 16 | -32 768 | 32 767 |
| `int` | 4 байти | 32 | -2 147 483 648 | 2 147 483 647 |
| `long long` | 8 байтів | 64 | ≈ -9.22e18 | ≈ 9.22e18 |
| `float` | 4 байти | 32 | ≈ 1.2e-38 | ≈ 3.4e38 |
| `double` | 8 байтів | 64 | ≈ 2.2e-308 | ≈ 1.8e308 |

## unsigned модифікатори

Беззнакові типи використовують той самий біт для знаку для збільшення діапазону:

```c++
unsigned int x;     // 0 до 4 294 967 295
unsigned short y;   // 0 до 65 535
```

## Стандартні гарантії

Хоча точні розміри залежать від платформи, C++ гарантує:

```c++
sizeof(char) ≤ sizeof(short) ≤ sizeof(int) ≤ sizeof(long) ≤ sizeof(long long)
sizeof(float) ≤ sizeof(double) ≤ sizeof(long double)
sizeof(char) == 1
```

## Чому це важливо?

- **Вибір типу впливає на продуктивність** — менший тип економить пам'ять
- **Переповнення** — якщо значення виходить за межі діапазону типу
- **Переносимість** — код може поводитися по-різному на різних платформах

## Правило

Завжди перевіряйте розміри типів на вашій платформі за допомогою `sizeof`, особливо при роботі з бінарними даними.
