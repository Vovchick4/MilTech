# Урок №38. Символьний тип даних char

`char` (від англ. character — символ) зберігає один символ. Внутрішньо він зберігається як ціле число (код символу за таблицею ASCII).

## Оголошення

```c++
char ch { 'a' };      // символ a
char digit { '5' };   // символ 5 (не число!)
char nl { '\n' };     // символ нового рядка
```

Символьні літерали завжди в одинарних лапках `''`.

## Таблиця ASCII

Кожен символ має числовий код 0-127 (стандартний ASCII):

```c++
char ch { 'A' };
std::cout << ch;              // виведе: A
std::cout << static_cast<int>(ch);  // виведе: 65 (код ASCII)
```

Основні символи ASCII:

| Діапазон | Символи |
|---------|---------|
| 65-90 | A-Z (латинські великі) |
| 97-122 | a-z (латинські малі) |
| 48-57 | 0-9 (цифри) |
| 0-31 | Керуючі символи |
| 32 | Пробіл |

## Escape-послідовності (керуючі символи)

```c++
char tab = '\t';      // табуляція
char newline = '\n';  // новий рядок
char backslash = '\\'; // зворотній слеш
char quote = '\'';    // одинарна лапка
char null_char = '\0'; // нульовий символ (null)
```

## char як ціле число

Оскільки `char` — це фактично ціле число, з ним можна виконувати арифметичні операції:

```c++
char ch { 'a' };
ch = ch + 1;          // стає 'b' (код 98)
std::cout << ch;      // b

char digit { '7' };
int value = digit - '0';   // перетворення символу цифри в число: 7
```

## signed char vs unsigned char

```c++
char c;          // може бути signed або unsigned — залежить від платформи
signed char sc;  // явно зі знаком (-128 до 127)
unsigned char uc; // явно без знаку (0 до 255)
```

## Введення символів

```c++
#include <iostream>

int main()
{
    char ch;
    std::cout << "Введіть символ: ";
    std::cin >> ch;      // читає один символ
    std::cout << "Ви ввели: " << ch << "\n";
    std::cout << "Код ASCII: " << static_cast<int>(ch) << "\n";
    return 0;
}
```

## Приклад: вивід всіх символів ASCII

```c++
#include <iostream>

int main()
{
    for (int i = 32; i <= 126; ++i)
    {
        std::cout << static_cast<char>(i) << " ";
        if ((i - 31) % 16 == 0) std::cout << "\n";
    }
    return 0;
}
```

## Правило

Використовуйте `char` для зберігання одного символу. Для рядків використовуйте `std::string`.
