# Урок №37. Логічний тип даних bool

`bool` (від англ. Boolean — логічний тип) може приймати лише два значення: `true` (істина) або `false` (хибність).

## Оголошення та ініціалізація

```c++
bool isReady { true };      // істина
bool isDone { false };      // хибність
bool flag {};               // false (value-ініціалізація)
```

## Виведення bool

За замовчуванням `std::cout` виводить `bool` як 0 або 1:

```c++
bool flag { true };
std::cout << flag;          // 1

bool done { false };
std::cout << done;          // 0
```

Щоб виводити `true` / `false` як текст:

```c++
#include <iostream>

int main()
{
    bool flag { true };
    std::cout << std::boolalpha;   // вмикаємо текстовий режим
    std::cout << flag;             // "true"
    
    std::cout << std::noboolalpha; // вимикаємо
    std::cout << flag;             // 1
    return 0;
}
```

## Перетворення в bool

Цілі числа можна конвертувати в `bool`:
- 0 → `false`
- будь-яке ненульове → `true`

```c++
bool a = 0;      // false
bool b = 1;      // true
bool c = -42;    // true
bool d = 5;      // true
```

## bool в умовних виразах

Найчастіше `bool` використовується в умовних операторах:

```c++
bool isStudent { true };

if (isStudent)
{
    std::cout << "Студент\n";
}
else
{
    std::cout << "Не студент\n";
}
```

## Логічні оператори, що повертають bool

```c++
int a = 5, b = 10;
bool result;

result = (a == b);    // false
result = (a != b);    // true
result = (a < b);     // true
result = (a > b);     // false
result = (a <= b);    // true
result = (a >= b);    // false

result = (a > 0) && (b > 0);   // true (AND)
result = (a > 0) || (b < 0);   // true (OR)
result = !(a == b);             // true (NOT)
```

## Функції, що повертають bool

```c++
#include <iostream>

bool isEven(int number)
{
    return number % 2 == 0;
}

int main()
{
    std::cout << std::boolalpha;
    std::cout << isEven(4);   // true
    std::cout << isEven(7);   // false
    return 0;
}
```

## Стиль: уникайте зайвих порівнянь

```c++
if (isReady == true)   // зайве
if (isReady)           // краще

if (isDone == false)   // зайве
if (!isDone)           // краще
```

## Правило

Використовуйте `bool` для логічних значень і прапорців (flags). Назви змінних формулюйте як питання: `isReady`, `hasValue`, `canProcess`.
