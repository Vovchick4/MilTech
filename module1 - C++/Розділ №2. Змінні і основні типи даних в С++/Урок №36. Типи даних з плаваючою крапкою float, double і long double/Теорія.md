# Урок №36. Типи даних з плаваючою крапкою: float, double і long double

Типи з плаваючою крапкою (floating-point types) зберігають числа з дробовою частиною.

## Основні типи

| Тип | Розмір | Точність | Приблизний діапазон |
|-----|--------|----------|-------------------|
| `float` | 4 байти | ~7 десяткових цифр | ±1.2e-38 до ±3.4e38 |
| `double` | 8 байтів | ~15 десяткових цифр | ±2.2e-308 до ±1.8e308 |
| `long double` | 8, 12 або 16 байтів | залежить від платформи | залежить від платформи |

## Оголошення та ініціалізація

```c++
float f { 3.14f };          // f суфікс для float
double d { 3.14 };          // за замовчуванням double
long double ld { 3.14L };   // L суфікс для long double
```

## Наукова нотація

```c++
double avogadro = 6.022e23;      // 6.022 × 10^23
double electron = 1.602e-19;     // 1.602 × 10^-19
double mass = 9.109e-31;         // маса електрона
```

## Точність і округлення

Типи з плаваючою крапкою мають обмежену точність:

```c++
#include <iostream>
#include <iomanip>

int main()
{
    float f { 123456789.0f };
    double d { 123456789.0 };
    
    std::cout << std::setprecision(20);
    std::cout << "float:   " << f << "\n";   // 123456792 (втрата точності)
    std::cout << "double:  " << d << "\n";   // 123456789 (точно)
    
    // Проблема порівняння
    double a = 0.1 + 0.2;
    std::cout << "0.1 + 0.2 = " << a << "\n";  // 0.30000000000000004
    return 0;
}
```

## Ніколи не порівнюйте напряму!

```c++
double a = 0.1 + 0.2;
if (a == 0.3)  // МОЖЕ НЕ СПРАЦЮВАТИ!
{
    // цей код може не виконатися
}

// Правильний підхід: перевіряйте різницю
double epsilon = 1e-10;  // допустима похибка
if (std::abs(a - 0.3) < epsilon)
{
    // OK
}
```

## Спеціальні значення

```c++
double inf = 1.0 / 0.0;       // +∞ (нескінченність)
double negInf = -1.0 / 0.0;   // -∞
double nan = 0.0 / 0.0;       // NaN (Not a Number)

std::cout << std::isinf(inf);    // true
std::cout << std::isnan(nan);    // true
```

## Рекомендації

1. За замовчуванням використовуйте `double` (він оптимальний на сучасних CPU)
2. `float` використовуйте тільки для економії пам'яті (тисячі елементів)
3. Ніколи не порівнюйте числа з плаваючою крапкою через `==`
4. Будьте обережні з накопиченням похибок у циклах

## Правило

`double` — стандартний вибір для роботи з дробовими числами. `float` — для економії пам'яті.
