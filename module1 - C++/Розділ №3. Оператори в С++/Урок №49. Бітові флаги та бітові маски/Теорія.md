# Урок №49. Бітові флаги та бітові маски

## Бітові флаги

`Бітовий флаг` — це окремий біт, який використовується як прапорець (true/false). Замість того, щоб мати 8 окремих змінних `bool`, можна використати один `unsigned char` (8 бітів).

## Бітові маски

`Бітова маска` — це число, яке використовується для вибору конкретних бітів.

### Визначення масок

```c++
#include <cstdint>

// Маски для різних прапорців
constexpr std::uint8_t IS_ADMIN   = 0b0000'0001;  // біт 0
constexpr std::uint8_t IS_ACTIVE  = 0b0000'0010;  // біт 1
constexpr std::uint8_t HAS_ACCESS = 0b0000'0100;  // біт 2
constexpr std::uint8_t IS_BANNED  = 0b0000'1000;  // біт 3
```

### Встановлення флагу (|)

```c++
std::uint8_t userFlags = 0;

userFlags |= IS_ADMIN;    // встановлюємо біт 0
userFlags |= IS_ACTIVE;   // встановлюємо біт 1
// userFlags = 0b0000'0011
```

### Перевірка флагу (&)

```c++
if (userFlags & IS_ADMIN)
{
    std::cout << "Користувач — адмін\n";
}

if (userFlags & IS_ACTIVE)
{
    std::cout << "Користувач активний\n";
}
```

### Скидання флагу (& ~)

```c++
userFlags &= ~IS_ACTIVE;  // скидаємо біт 1 (вимикаємо)
```

### Перемикання флагу (^)

```c++
userFlags ^= IS_BANNED;   // перемикаємо біт 3: 0→1 або 1→0
```

## Повний приклад

```c++
#include <iostream>
#include <bitset>
#include <cstdint>

constexpr std::uint8_t IS_ADMIN   = 0b0000'0001;
constexpr std::uint8_t IS_ACTIVE  = 0b0000'0010;
constexpr std::uint8_t HAS_ACCESS = 0b0000'0100;
constexpr std::uint8_t IS_BANNED  = 0b0000'1000;

void printFlags(std::uint8_t flags)
{
    std::cout << "Флаги: " << std::bitset<8>(flags) << "\n";
    std::cout << "  Адмін: "   << ((flags & IS_ADMIN) ? "Так" : "Ні") << "\n";
    std::cout << "  Активний: " << ((flags & IS_ACTIVE) ? "Так" : "Ні") << "\n";
    std::cout << "  Доступ: "  << ((flags & HAS_ACCESS) ? "Так" : "Ні") << "\n";
    std::cout << "  Заблокован: " << ((flags & IS_BANNED) ? "Так" : "Ні") << "\n";
}

int main()
{
    std::uint8_t user = IS_ACTIVE | HAS_ACCESS;
    printFlags(user);
    
    user |= IS_ADMIN;
    printFlags(user);
    
    user &= ~HAS_ACCESS;
    printFlags(user);
    
    return 0;
}
```

## Коли використовувати?

| Підхід | Коли використовувати |
|--------|---------------------|
| `bool` змінні | Мало прапорців (1-3), простота важливіша |
| Бітові флаги | Багато прапорців, економія пам'яті, мережеві протоколи |

## Альтернатива: std::bitset

```c++
#include <bitset>

std::bitset<8> flags;
flags.set(0);        // встановити біт 0
flags.test(0);       // перевірити біт 0 (повертає bool)
flags.reset(1);      // скинути біт 1
flags.flip(2);       // перемкнути біт 2
```

## Правило

Для бітових операцій завжди використовуйте `unsigned` типи. Знаковий зсув для від'ємних чисел — невизначена поведінка.
