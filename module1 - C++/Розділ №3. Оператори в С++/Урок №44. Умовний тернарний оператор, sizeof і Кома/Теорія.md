# Урок №44. Умовний тернарний оператор, sizeof і Кома

## Тернарний оператор (?:)

`Умовний тернарний оператор` — це скорочена форма if/else, яка повертає значення:

```c++
результат = (умова) ? значення_якщо_true : значення_якщо_false;
```

### Приклад

```c++
int max = (a > b) ? a : b;   // max = більше з a та b

// Еквівалентно:
int max;
if (a > b)
    max = a;
else
    max = b;
```

### Використання

```c++
std::cout << (isEven ? "Парне" : "Непарне") << "\n";

int x = 5;
int y = (x > 0) ? x : 0;   // y = x, якщо x > 0, інакше 0

int absValue = (x < 0) ? -x : x;  // модуль числа
```

### Вкладені тернарні оператори (обережно!)

```c++
// Складно читати:
int result = (a > b) ? ((a > c) ? a : c) : ((b > c) ? b : c);

// Краще розбити:
int max = (a > b) ? a : b;
max = (max > c) ? max : c;
```

### Правило

Тернарний оператор добре підходить для простих умов. Для складних — використовуйте if/else.

## sizeof

`sizeof` повертає розмір типу (або змінної) в байтах:

```c++
sizeof(int)       // 4
sizeof(double)    // 8
sizeof(arr)       // загальний розмір масиву в байтах

int x;
sizeof x          // 4 (дужки необов'язкові для змінних)
sizeof(int)       // 4 (для типів дужки обов'язкові)
```

### Корисне використання: кількість елементів масиву

```c++
int arr[] = {1, 2, 3, 4, 5};
int count = sizeof(arr) / sizeof(arr[0]);  // 5
// В C++17 краще: std::size(arr);
```

## Оператор Кома (,)

Оператор `,` дозволяє виконати кілька виразів в одному місці. Повертає значення останнього виразу:

```c++
int x = (1, 2, 3);    // x = 3 (значення останнього виразу)

int a = 5, b = 10;    // це НЕ оператор кома, а роздільник оголошень

// for-цикл з кількома лічильниками
for (int i = 0, j = 10; i < j; ++i, --j)
{
    std::cout << i << " " << j << "\n";
}
```

### Небезпека: кома vs крапка з комою

```c++
// Плутанина з комою в умові
if (x > 0, y > 0)   // виконується x > 0 (ігнорується), потім y > 0
                     // результат залежить ТІЛЬКИ від y > 0!

// Правильно:
if (x > 0 && y > 0) // логічне І
```

## Приклад використання

```c++
#include <iostream>

int main()
{
    // Тернарний оператор
    int a = 7, b = 3;
    std::cout << "Максимум: " << ((a > b) ? a : b) << "\n";
    
    // sizeof
    int arr[] = {10, 20, 30, 40};
    std::cout << "Елементів: " << sizeof(arr) / sizeof(arr[0]) << "\n";
    
    return 0;
}
```
