# Урок №48. Побітові оператори

Побітові оператори працюють безпосередньо з бітами цілих чисел.

## Таблиця операторів

| Оператор | Назва | Приклад |
|----------|-------|---------|
| `~` | Побітове НЕ (NOT) | `~x` |
| `&` | Побітове І (AND) | `x & y` |
| `\|` | Побітове АБО (OR) | `x \| y` |
| `^` | Побітове XOR (виключне АБО) | `x ^ y` |
| `<<` | Зсув вліво | `x << n` |
| `>>` | Зсув вправо | `x >> n` |

## Побітове І (&)

1 тільки якщо ОБИДВА біти 1:

```
  0110  (6)
& 0101  (5)
--------
  0100  (4)
```

```c++
int x = 6 & 5;  // 4
```

Використання: перевірка, чи встановлений біт:

```c++
if (flags & FLAG_MASK)  // перевіряємо біт
```

## Побітове АБО (|)

1 якщо ХОЧА Б ОДИН біт 1:

```
  0110  (6)
| 0101  (5)
--------
  0111  (7)
```

```c++
int x = 6 | 5;  // 7
```

Використання: встановлення біта:

```c++
flags = flags | FLAG_MASK;   // встановлюємо біт
flags |= FLAG_MASK;          // скорочена форма
```

## Побітове XOR (^)

1 якщо біти РІЗНІ:

```
  0110  (6)
^ 0101  (5)
--------
  0011  (3)
```

Властивість: `x ^ x == 0`, `x ^ 0 == x`.

```c++
int x = 6 ^ 5;  // 3
```

## Побітове НЕ (~)

Інвертує всі біти (0→1, 1→0):

```
~ 0110  (6)
--------
  1001  (9 на 4 бітах, але на 32 бітах це 4294967289)
```

## Зсув вліво (<<) і вправо (>>)

Зсуває біти вліво/вправо:

```c++
int x = 3;       // 0011
int y = x << 2;  // 1100 = 12 (зсув вліво = множення на 2ⁿ)
int z = x >> 1;  // 0001 = 1 (зсув вправо = ділення на 2ⁿ)

// Приклад:
3 << 2  == 3 * 4  == 12
8 >> 2  == 8 / 4  == 2
```

## Приклад використання

```c++
#include <iostream>
#include <bitset>

int main()
{
    unsigned int a = 0b1100;
    unsigned int b = 0b1010;
    
    std::cout << "a = " << std::bitset<4>(a) << " (" << a << ")\n";
    std::cout << "b = " << std::bitset<4>(b) << " (" << b << ")\n";
    std::cout << "a & b = " << std::bitset<4>(a & b) << "\n";
    std::cout << "a | b = " << std::bitset<4>(a | b) << "\n";
    std::cout << "a ^ b = " << std::bitset<4>(a ^ b) << "\n";
    std::cout << "~a    = " << std::bitset<4>(~a) << "\n";
    std::cout << "a << 1 = " << std::bitset<4>(a << 1) << "\n";
    std::cout << "a >> 1 = " << std::bitset<4>(a >> 1) << "\n";
    
    return 0;
}
```

## Правила

1. Використовуйте `unsigned` типи для побітових операцій (знаковий зсув — невизначена поведінка)
2. Не плутайте `&` (побітове) та `&&` (логічне)
3. Зсув вліво заповнює нулями, зсув вправо для unsigned — нулями
