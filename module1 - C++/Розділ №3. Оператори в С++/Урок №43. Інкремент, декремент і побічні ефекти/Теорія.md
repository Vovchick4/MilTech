# Урок №43. Інкремент, декремент і побічні ефекти

## Інкремент (++)

Збільшує значення змінної на 1:

```c++
int x = 5;
x++;  // x = x + 1 → x = 6
++x;  // x = x + 1 → x = 7
```

## Декремент (--)

Зменшує значення змінної на 1:

```c++
int x = 5;
x--;  // x = x - 1 → x = 4
--x;  // x = x - 1 → x = 3
```

## Префіксний vs Постфіксний

**Префіксний** (`++x`): значення збільшується, потім повертається нове значення.

**Постфіксний** (`x++`): повертається старе значення, потім значення збільшується.

```c++
int x = 5;
int a = ++x;  // x = 6, a = 6 (збільшили, потім присвоїли)

int y = 5;
int b = y++;  // b = 5, y = 6 (присвоїли старе, потім збільшили)
```

### Приклад з циклом

```c++
int i = 0;
while (i < 3)
{
    std::cout << i++ << " ";  // постфікс: 0 1 2
}
// i = 3

int j = 0;
while (j < 3)
{
    std::cout << ++j << " ";  // префікс: 1 2 3
}
// j = 3
```

## Побічні ефекти

`Побічний ефект` — коли операція змінює стан програми (наприклад, значення змінної) крім повернення результату.

```c++
int x = 5;
int y = x++ + 2;   // побічний ефект: x змінюється на 6
```

### Небезпечний код: невизначена поведінка

Не використовуйте змінну більше одного разу у виразі, якщо вона змінюється:

```c++
int x = 5;
int y = x++ + x;   // НЕВИЗНАЧЕНО! C++ не гарантує порядок обчислення

int arr[] = {1, 2, 3};
int i = 0;
arr[i] = i++;      // НЕВИЗНАЧЕНО!
```

## Продуктивність

Для простих типів (`int`, `double`) різниці немає. Для складних типів (ітератори) префікс `++x` ефективніший, оскільки постфікс створює копію:

```c++
// Префікс — ефективніший для складних типів
auto it = vec.begin();
++it;  // змінює ітератор без копіювання
it++;  // створює копію старого стану
```

## Правила

1. Для простих типів використовуйте префікс `++x` за замовчуванням (звичка)
2. Ніколи не використовуйте `++` зі змінною більше одного разу у виразі
3. Використовуйте окремі рядки для інкременту: `x++;` замість `y = x++ + 2;`

## Приклад

```c++
#include <iostream>

int main()
{
    int x = 5;
    
    std::cout << "x   = " << x << "\n";    // 5
    std::cout << "++x = " << ++x << "\n";  // 6 (префікс: збільшити, потім вивести)
    
    x = 5;
    std::cout << "x++ = " << x++ << "\n";  // 5 (постфікс: вивести, потім збільшити)
    std::cout << "x   = " << x << "\n";    // 6
    
    return 0;
}
```
