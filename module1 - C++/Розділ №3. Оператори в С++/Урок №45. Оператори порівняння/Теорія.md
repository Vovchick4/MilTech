# Урок №45. Оператори порівняння

Оператори порівняння порівнюють два значення і повертають `bool` (`true` або `false`).

## Список операторів

| Оператор | Значення | Приклад | Результат |
|----------|----------|---------|-----------|
| `==` | дорівнює | `5 == 3` | `false` |
| `!=` | не дорівнює | `5 != 3` | `true` |
| `<` | менше | `5 < 3` | `false` |
| `>` | більше | `5 > 3` | `true` |
| `<=` | менше або дорівнює | `5 <= 5` | `true` |
| `>=` | більше або дорівнює | `5 >= 3` | `true` |

## Важлива відмінність: = vs ==

Найпоширеніша помилка початківців — плутанина присвоювання та порівняння:

```c++
int x = 5;

if (x = 10)      // ХІБА! це присвоювання, x стає 10, умова завжди true
{
    // виконується завжди!
}

if (x == 10)     // ПРАВИЛЬНО: порівняння
{
    // виконується тільки якщо x == 10
}
```

## Порівняння чисел з плаваючою крапкою

Через обмежену точність, числа з плаваючою крапкою не можна порівнювати напряму:

```c++
double a = 0.1 + 0.2;  // 0.30000000000000004
if (a == 0.3)          // МОЖЕ НЕ СПРАЦЮВАТИ!
{
    // цей код може не виконатися!
}

// Правильний підхід:
const double EPSILON = 1e-9;
if (std::abs(a - 0.3) < EPSILON)  // перевіряємо, чи різниця достатньо мала
{
    // OK
}
```

## Порівняння символів

Символи порівнюються за ASCII-кодами:

```c++
'a' < 'b'    // true (97 < 98)
'A' < 'a'    // true (65 < 97)
'0' < '9'    // true (48 < 57)
```

## Ланцюжкові порівняння

У C++ не можна писати математичні ланцюжки:

```c++
// Математика: 5 < x < 10
if (5 < x < 10)   // НЕПРАВИЛЬНО! Це (5 < x) < 10 → true/false < 10 → завжди true

// Правильно:
if (x > 5 && x < 10)  // ПРАВИЛЬНО
```

## Порівняння рядків (std::string)

```c++
std::string s1 = "apple";
std::string s2 = "banana";

s1 == s2   // false
s1 != s2   // true
s1 < s2    // true (лексикографічне порівняння)
```

## Приклад

```c++
#include <iostream>

int main()
{
    int a = 5, b = 10;
    
    std::cout << std::boolalpha;
    std::cout << "a == b: " << (a == b) << "\n";  // false
    std::cout << "a != b: " << (a != b) << "\n";  // true
    std::cout << "a < b:  " << (a < b) << "\n";   // true
    std::cout << "a > b:  " << (a > b) << "\n";   // false
    
    return 0;
}
```
