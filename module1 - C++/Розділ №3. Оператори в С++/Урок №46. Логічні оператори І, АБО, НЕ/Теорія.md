# Урок №46. Логічні оператори: І, АБО, НЕ

Логічні оператори працюють з `bool` значеннями і повертають `bool`.

## Логічне І (&&)

Повертає `true`, тільки якщо ОБИДВА операнди `true`:

| A | B | A && B |
|---|---|--------|
| false | false | false |
| false | true | false |
| true | false | false |
| true | true | **true** |

```c++
bool a = true, b = false;
bool result = a && b;  // false

if (x > 0 && x <= 100)  // x в діапазоні (0, 100]
{
    // OK
}
```

## Логічне АБО (||)

Повертає `true`, якщо ХОЧА Б ОДИН операнд `true`:

| A | B | A \|\| B |
|---|---|---------|
| false | false | false |
| false | true | **true** |
| true | false | **true** |
| true | true | **true** |

```c++
bool a = true, b = false;
bool result = a || b;  // true

if (x < 0 || x > 100)  // x за межами діапазону
{
    // помилка
}
```

## Логічне НЕ (!)

Інвертує значення: `true` → `false`, `false` → `true`:

```c++
bool flag = true;
bool result = !flag;  // false

if (!isReady)  // якщо НЕ готово
{
    // чекаємо
}
```

## Скорочене обчислення (short-circuit evaluation)

C++ використовує скорочене обчислення — якщо результат вже відомий, інша частина не обчислюється:

```c++
if (ptr != nullptr && *ptr == 5)  // безпечно! *ptr не обчислюється, якщо ptr == nullptr
{
    // ...
}

if (x > 0 || expensiveFunc())  // expensiveFunc() не викликається, якщо x > 0
{
    // ...
}
```

## Пріоритет логічних операторів

```c++
! (найвищий)
&&
|| (найнижчий)
```

```c++
if (a || b && c)     // a || (b && c) — && має вищий пріоритет
if ((a || b) && c)   // інша логіка — використовуйте дужки!
```

## Поєднання операторів

```c++
bool isLeapYear(int year)
{
    // Рік високосний, якщо:
    // ділиться на 4, не ділиться на 100, АБО ділиться на 400
    return (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
}
```

## Приклад

```c++
#include <iostream>

int main()
{
    int age;
    bool hasParentConsent;
    
    std::cout << "Введіть вік: ";
    std::cin >> age;
    
    bool canRegister = (age >= 18) || (age >= 14 && hasParentConsent);
    
    if (canRegister)
        std::cout << "Реєстрація дозволена\n";
    else
        std::cout << "Реєстрація заборонена\n";
    
    return 0;
}
```

## Правило

Використовуйте дужки, щоб зробити логіку зрозумілою, навіть якщо знаєте пріоритет.
