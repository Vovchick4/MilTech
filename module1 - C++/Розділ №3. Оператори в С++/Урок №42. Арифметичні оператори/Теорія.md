# Урок №42. Арифметичні оператори

Арифметичні оператори виконують математичні операції над числами.

## Бінарні арифметичні оператори

| Оператор | Назва | Приклад | Результат |
|----------|-------|---------|-----------|
| `+` | додавання | `5 + 3` | 8 |
| `-` | віднімання | `5 - 3` | 2 |
| `*` | множення | `5 * 3` | 15 |
| `/` | ділення | `5 / 3` | 1 (int) |
| `%` | остача (модуль) | `5 % 3` | 2 |

## Цілочисельне ділення

Ділення двох цілих чисел дає цілий результат (дробова частина відкидається):

```c++
int a = 7 / 2;     // 3 (не 3.5!)
int b = 7 / 3;     // 2
int c = 7 / 7;     // 1
int d = 7 / 8;     // 0
```

Щоб отримати дробовий результат, хоча б один операнд має бути дробовим:

```c++
double e = 7.0 / 2;    // 3.5
double f = 7 / 2.0;    // 3.5
double g = (double)7 / 2;  // 3.5 (явне приведення)
double h = 7 / 2;      // 3.0 (спочатку ціле, потім конвертація)
```

## Оператор остачі (%)

`%` повертає остачу від ділення:

```c++
int a = 7 % 3;   // 1 (7 = 3*2 + 1)
int b = 10 % 4;  // 2
int c = 8 % 2;   // 0 (8 парне)
int d = 9 % 2;   // 1 (9 непарне)
```

### Перевірка парності

```c++
bool isEven(int n)
{
    return n % 2 == 0;  // true якщо парне
}
```

### Важливо: % не працює з float/double

```c++
double x = 5.5 % 2.0;  // ПОМИЛКА! % тільки для цілих чисел
double x = std::fmod(5.5, 2.0);  // OK: використовуйте fmod
```

## Ділення на нуль

```c++
int x = 5 / 0;   // НЕВИЗНАЧЕНА ПОВЕДІНКА (аварійне завершення)
double y = 5.0 / 0.0;  // OK: y = inf (нескінченність)
```

## Унарні оператори

```c++
int x = 5;
int y = -x;      // -5 (унарний мінус)
int z = +x;      // 5 (унарний плюс, рідко використовується)
```

## Побічні ефекти

Арифметичні оператори не змінюють свої операнди (на відміну від `+=`):

```c++
int a = 5;
int b = a + 3;   // a все ще 5, b = 8
a = a + 3;       // a = 8 (явне присвоювання)
a += 3;          // a = 11 (compound assignment)
```

## Приклад: середнє арифметичне

```c++
#include <iostream>

int main()
{
    int a = 10, b = 15, c = 20;
    double average = (a + b + c) / 3.0;  // важливо: 3.0!
    std::cout << "Середнє: " << average;
    return 0;
}
```

## Правило

При діленні цілих чисел, якщо потрібен дробовий результат — конвертуйте хоча б один операнд у `double`.

## Тест

1. Чому `7 / 2` дорівнює 3, а не 3.5?
2. Для яких типів даних працює оператор `%`?
3. Як обчислити середнє арифметичне трьох цілих чисел з дробовою частиною?
