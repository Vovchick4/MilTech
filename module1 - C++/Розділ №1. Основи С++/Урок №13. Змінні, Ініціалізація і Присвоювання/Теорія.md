# Урок №13. Змінні, Ініціалізація і Присвоювання

### Змінні

`Змінна` — це іменована ділянка пам'яті, яка зберігає певне значення. У C++ кожна змінна повинна мати тип, який визначає, які дані вона може зберігати.

```c++
int x;          // оголошення змінної x цілочисельного типу
double y;       // оголошення змінної y з плаваючою крапкою
char letter;    // оголошення символьної змінної
```

### Ініціалізація

`Ініціалізація` — це надання змінній початкового значення при її створенні. В C++ існує кілька форм ініціалізації:

```c++
int a = 5;          // копіювальна ініціалізація (copy initialization)
int b(5);           // пряма ініціалізація (direct initialization)
int c { 5 };        // uniform-ініціалізація (рекомендована)
int d {};           // value-ініціалізація (нульове значення)
```

Uniform-ініціалізація `{}` є найбезпечнішою, оскільки не дозволяє звужувальних конвертацій:

```c++
int x { 5.5 };  // помилка! double не може бути неявно конвертований в int
int y = 5.5;    // OK, але значення буде 5 (втрата даних)
```

### Присвоювання

`Присвоювання` — це надання нового значення вже існуючій змінній:

```c++
int x { 5 };    // ініціалізація
x = 7;          // присвоювання — змінна x тепер має значення 7
```

### Неініціалізовані змінні

Використання неініціалізованої змінної призводить до невизначеної поведінки:

```c++
int x;
std::cout << x; // НЕ БЕЗПЕЧНО! x містить "сміття"
```

### Правило: Завжди ініціалізуйте змінні при оголошенні.
