# Урок №14. Об'єкти cout, cin і endl

### std::cout — виведення даних

`std::cout` (character output) — це об'єкт з бібліотеки `iostream`, який виводить дані на консоль. Використовується з оператором `<<` (вставки):

```c++
#include <iostream>

int main()
{
    std::cout << "Hello, world!";
    std::cout << 42;
    std::cout << "Значення: " << 5 + 3;
    return 0;
}
```

Можна об'єднувати кілька виведень в одному рядку:

```c++
std::cout << "X = " << x << ", Y = " << y;
```

### std::endl — кінець рядка

`std::endl` (end line) вставляє символ нового рядка і скидає буфер виведення:

```c++
std::cout << "Рядок 1" << std::endl;
std::cout << "Рядок 2" << std::endl;
```

Альтернатива — символ `\n` (швидший, не скидає буфер):

```c++
std::cout << "Рядок 1\n";
std::cout << "Рядок 2\n";
```

### std::cin — введення даних

`std::cin` (character input) — об'єкт для отримання даних від користувача. Використовується з оператором `>>` (вилучення):

```c++
#include <iostream>

int main()
{
    int x;
    std::cout << "Введіть число: ";
    std::cin >> x;
    std::cout << "Ви ввели: " << x << std::endl;
    return 0;
}
```

Можна вводити кілька значень:

```c++
int a, b;
std::cin >> a >> b;  // ввід двох чисел через пробіл
```

### Важливі зауваження

- `>>` пропускає пробіли, табуляції та символи нового рядка
- При некоректному вводі `std::cin` переходить у стан помилки
- Завжди підключайте `#include <iostream>` для використання цих об'єктів
