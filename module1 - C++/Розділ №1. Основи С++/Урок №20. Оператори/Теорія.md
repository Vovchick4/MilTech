# Урок №20. Оператори

`Оператор` — це спеціальний символ (або група символів), який виконує операцію над одним або кількома операндами.

### Унарні, бінарні та тернарні оператори

- `Унарні` — працюють з одним операндом: `-x`, `++x`, `!x`
- `Бінарні` — працюють з двома операндами: `a + b`, `x = 5`, `a == b`
- `Тернарний` — працює з трьома: `x ? y : z`

### Арифметичні оператори

```c++
int a = 5 + 3;    // + додавання, a = 8
int b = 5 - 3;    // - віднімання, b = 2
int c = 5 * 3;    // * множення, c = 15
int d = 5 / 3;    // / ділення (ціле!), d = 1
double e = 5.0 / 3.0; // e ≈ 1.666
int f = 5 % 3;    // % остача від ділення, f = 2
```

### Оператори присвоювання

```c++
int x = 5;        // = присвоювання
x += 3;           // x = x + 3
x -= 2;           // x = x - 2
x *= 4;           // x = x * 4
x /= 2;           // x = x / 2
x %= 3;           // x = x % 3
```

### Оператори порівняння (повертають true/false)

```c++
a == b   // дорівнює
a != b   // не дорівнює
a < b    // менше
a > b    // більше
a <= b   // менше або дорівнює
a >= b   // більше або дорівнює
```

### Логічні оператори

```c++
bool r1 = (a > 0) && (b > 0);  // && — логічне І (AND)
bool r2 = (a > 0) || (b > 0);  // || — логічне АБО (OR)
bool r3 = !(a > 0);             // ! — логічне НЕ (NOT)
```

### Пріоритет операцій

Пріоритет визначає порядок виконання:

```c++
int x = 5 + 3 * 2;   // 5 + (3 * 2) = 11, множення має вищий пріоритет
```
