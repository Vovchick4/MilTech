# Урок №28. Розробка ваших перших програм

### Процес розробки програми

1. **Визначте мету** — що програма повинна робити?
2. **Спроектуйте рішення** — які функції, змінні, структури даних потрібні?
3. **Напишіть код** — реалізуйте ваше рішення
4. **Скомпілюйте** — перевірте, чи код компілюється без помилок
5. **Протестуйте** — чи працює програма, як очікувалося?
6. **Налагодьте** — виправте помилки, якщо програма не працює коректно

### Приклад: програма-калькулятор

**Крок 1:** Мета — створити простий калькулятор, який додає два числа.

**Крок 2:** Дизайн — функція `main()` для вводу/виводу та функція `add()` для обчислення.

**Крок 3:** Код:

```c++
#include <iostream>

int add(int a, int b)
{
    return a + b;
}

int main()
{
    std::cout << "Введіть два числа: ";
    int x {}, y {};
    std::cin >> x >> y;

    std::cout << "Сума: " << add(x, y) << std::endl;
    return 0;
}
```

### Поради для початківців

1. **Починайте з малого** — спочатку змусьте працювати простий варіант
2. **Компілюйте часто** — не пишіть 100 рядків без компіляції
3. **Виправляйте першу помилку** — часто одна помилка спричиняє інші
4. **Тестуйте поступово** — перевіряйте кожен новий шматок коду
5. **Використовуйте дебагер** — не просто "вгадуйте" проблему

### Покрокове додавання функціоналу

```c++
// Версія 1: просто виводить текст
#include <iostream>
int main() {
    std::cout << "Калькулятор\n";
    return 0;
}

// Версія 2: ввід чисел
#include <iostream>
int main() {
    int x {}, y {};
    std::cin >> x >> y;
    return 0;
}

// Версія 3: додавання і вивід
#include <iostream>
int add(int a, int b) { return a + b; }
int main() {
    int x {}, y {};
    std::cin >> x >> y;
    std::cout << add(x, y);
    return 0;
}
```

### Типові помилки початківців

- Забута крапка з комою `;`
- Неініціалізовані змінні
- Плутанина `=` (присвоювання) і `==` (порівняння)
- Вихід за межі масиву
- Неправильний тип даних
