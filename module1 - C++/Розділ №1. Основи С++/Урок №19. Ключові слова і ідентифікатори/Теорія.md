# Урок №19. Ключові слова і ідентифікатори

### Ідентифікатори

`Ідентифікатор` — це ім'я, яке дається змінній, функції, класу тощо. В C++ діють правила для імен:

- Можуть містити: літери (a-z, A-Z), цифри (0-9), символ підкреслення `_`
- Не можуть починатися з цифри
- Чутливі до регістру (`myVar` ≠ `myvar`)
- Не можуть збігатися з ключовими словами

```c++
int value;      // OK
int _value;     // OK, але уникайте _ на початку (резервовано)
int 2value;     // ПОМИЛКА! починається з цифри
int my value;   // ПОМИЛКА! пробіл
int int;        // ПОМИЛКА! int — ключове слово
```

### Ключові слова

`Ключові слова` (keywords) — це зарезервовані слова мови C++, які мають спеціальне значення. Їх не можна використовувати як ідентифікатори.

Основні ключові слова C++:

| Категорія | Ключові слова |
|-----------|--------------|
| Типи даних | `int`, `double`, `char`, `bool`, `void`, `float` |
| Управління | `if`, `else`, `switch`, `case`, `break`, `default`, `for`, `while`, `do`, `continue`, `return`, `goto` |
| Модифікатори | `const`, `static`, `unsigned`, `signed`, `short`, `long` |
| Класи/ООП | `class`, `struct`, `public`, `private`, `protected`, `virtual`, `friend`, `this` |
| Інші | `namespace`, `using`, `typedef`, `sizeof`, `true`, `false`, `nullptr` |

Повний список ключових слів залежить від стандарту C++ (C++11, C++17, C++20 тощо).

### Конвенції іменування

Хоча C++ дозволяє різні стилі, важливо дотримуватися послідовності:

```c++
int studentAge;        // camelCase — змінні
void calculateGrade(); // camelCase — функції
const int MAX_SIZE;    // UPPER_CASE — константи
class StudentRecord;   // PascalCase — класи
```

### Правило: Використовуйте осмислені імена, які пояснюють призначення змінної.
