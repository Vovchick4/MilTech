# Урок №16. Параметри і аргументи функцій

### Параметри vs Аргументи

`Параметр` — це змінна у визначенні функції, яка приймає вхідне значення. `Аргумент` — це фактичне значення, яке передається у функцію при виклику.

```c++
void printSum(int a, int b)  // a і b — параметри
{
    std::cout << a + b;
}

int main()
{
    printSum(5, 3);  // 5 і 3 — аргументи
    return 0;
}
```

### Параметри за замовчуванням

Функція може мати значення за замовчуванням для параметрів:

```c++
void greet(std::string name = "гість")
{
    std::cout << "Привіт, " << name << "!";
}

greet();           // "Привіт, гість!"
greet("Олексій");  // "Привіт, Олексій!"
```

Параметри за замовчуванням повинні бути крайніми справа:

```c++
void func(int a, int b = 0, int c = 0);  // OK
void func(int a = 0, int b);             // ПОМИЛКА
```

### Передача за значенням

За замовчуванням аргументи передаються `за значенням` — функція отримує копію:

```c++
void inc(int x)
{
    x = x + 1;  // змінюється лише копія
}

int main()
{
    int a = 5;
    inc(a);
    std::cout << a;  // 5 (оригінал не змінився)
    return 0;
}
```

### Скільки параметрів може мати функція?

C++ не обмежує кількість параметрів, але для читабельності рекомендується не більше 4-5.
