# Урок №15. Функції

`Функція` — це іменований блок коду, який виконує певне завдання. Функції дозволяють структурувати програму, уникати дублювання коду та полегшувати читання.

### Структура функції

Функція складається з заголовка та тіла:

```c++
int add(int a, int b)  // заголовок: тип_повернення назва(параметри)
{                      // початок тіла
    return a + b;      // тіло: код, який виконує функція
}                      // кінець тіла
```

- `int` — тип значення, яке функція повертає
- `add` — ім'я функції
- `int a, int b` — параметри (вхідні дані)
- `return a + b` — повернення результату

### Функція main()

Головна функція `main()` — точка входу в програму. Виконання починається з неї:

```c++
int main()
{
    // код програми
    return 0;  // 0 означає успішне завершення
}
```

### Приклад виклику функції

```c++
#include <iostream>

int add(int a, int b)
{
    return a + b;
}

int main()
{
    int result = add(3, 4);  // виклик функції, result = 7
    std::cout << result;
    return 0;
}
```

### Тип void

Функція може нічого не повертати — використовуйте тип `void`:

```c++
void printHello()
{
    std::cout << "Hello!";
    // return не обов'язковий
}
```
