# Урок №3. Розширені можливості CMake

Тестування, встановлення, опції, змінні та інше.

## Додавання тестів (CTest)

```cmake
enable_testing()

add_test(NAME test_name COMMAND test_executable)

# З GTest
include(FetchContent)
FetchContent_Declare(
    googletest
    GIT_REPOSITORY https://github.com/google/googletest.git
    GIT_TAG release-1.12.0
)
FetchContent_MakeAvailable(googletest)

add_executable(mytests test_main.cpp)
target_link_libraries(mytests PRIVATE GTest::gtest GTest::gtest_main)
gtest_discover_tests(mytests)
```

## Інсталяція

```cmake
# Встановити виконуваний файл
install(TARGETS myapp RUNTIME DESTINATION bin)

# Встановити бібліотеку
install(TARGETS mylib
    LIBRARY DESTINATION lib
    ARCHIVE DESTINATION lib
)

# Встановити заголовки
install(DIRECTORY include/ DESTINATION include)
```

## Опції (змінні)

```cmake
# Опція
option(BUILD_TESTS "Build tests" ON)
option(USE_OPENMP "Use OpenMP" OFF)

if(BUILD_TESTS)
    enable_testing()
    add_subdirectory(tests)
endif()

if(USE_OPENMP)
    find_package(OpenMP)
    target_link_libraries(myapp PRIVATE OpenMP::OpenMP_CXX)
endif()
```

## Змінні

```cmake
# Користувацькі змінні
```

## Generator expressions

```cmake
# Умовні вирази при збірці
target_compile_definitions(myapp PRIVATE
    $<$<CONFIG:Debug>:DEBUG_MODE>
    $<$<CONFIG:Release>:NDEBUG>
)

# В залежності від платформи
target_sources(myapp PRIVATE
    $<$<PLATFORM_ID:Windows>:windows_impl.cpp>
    $<$<PLATFORM_ID:Linux>:linux_impl.cpp>
)
```

## Приклад: повний проект

```cmake
cmake_minimum_required(VERSION 3.10)
project(AdvancedApp)

# Опції
option(BUILD_TESTS "Build tests" ON)
option(USE_SANITIZERS "Enable sanitizers" OFF)

# Стандарт
set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

# Бібліотека
add_library(core STATIC src/core.cpp include/core.h)
target_include_directories(core PUBLIC include)

# Сананітайзери
if(USE_SANITIZERS)
    target_compile_options(core PRIVATE -fsanitize=address)
    target_link_options(core PRIVATE -fsanitize=address)
endif()

# Головний додаток
add_executable(myapp src/main.cpp)
target_link_libraries(myapp PRIVATE core)

# Тести
if(BUILD_TESTS)
    enable_testing()
    add_subdirectory(tests)
endif()

# Інсталяція
install(TARGETS myapp core RUNTIME DESTINATION bin LIBRARY DESTINATION lib)
install(DIRECTORY include/ DESTINATION include)
```

## Підказки

```cmake
# Заборонити in-source збірку
if(${CMAKE_SOURCE_DIR} STREQUAL ${CMAKE_BINARY_DIR})
    message(FATAL_ERROR "In-source build is not allowed!")
endif()

# Використовувати ccache
find_program(CCACHE_PROGRAM ccache)
if(CCACHE_PROGRAM)
    set(CMAKE_CXX_COMPILER_LAUNCHER ${CCACHE_PROGRAM})
endif()
```

## Правило

Використовуйте опції для гнучкості, Generator Expressions для платформозалежного коду, CTest для автоматизації тестів.
