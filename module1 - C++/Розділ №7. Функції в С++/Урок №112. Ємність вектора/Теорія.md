# Урок №112. Ємність вектора

`std::vector` має два важливі атрибути: розмір (size) та ємність (capacity).

## size vs capacity

```c++
std::vector<int> v;
v.reserve(10);       // ємність = 10

std::cout << v.size();      // 0 — реальна кількість елементів
std::cout << v.capacity();  // 10 — скільки місця зарезервовано

v.push_back(1);      // size = 1, capacity = 10
v.push_back(2);      // size = 2, capacity = 10
```

## Як vector росте?

Коли size досягає capacity, вектор:
1. Виділяє НОВИЙ більший блок пам'яті (зазвичай ×2)
2. Копіює/переміщує всі елементи
3. Звільняє старий блок

```c++
std::vector<int> v;
for (int i = 0; i < 10; ++i)
{
    v.push_back(i);
    std::cout << "Size: " << v.size() 
              << ", Capacity: " << v.capacity() << "\n";
}
// Типовий вивід:
// Size: 1, Capacity: 1
// Size: 2, Capacity: 2
// Size: 3, Capacity: 4
// Size: 4, Capacity: 4
// Size: 5, Capacity: 8
// ...
```

## reserve() — завчасне виділення

Якщо ви знаєте приблизну кількість елементів, використовуйте `reserve()`:

```c++
std::vector<int> v;
v.reserve(1000);  // одне виділення

for (int i = 0; i < 1000; ++i)
    v.push_back(i);  // без перевиділень
```

## shrink_to_fit() — звільнення зайвої пам'яті

```c++
std::vector<int> v(1000);
v.clear();  // size = 0, але capacity все ще 1000
v.shrink_to_fit();  // capacity = 0 (або мінімальне)
```

## resize() — зміна розміру

```c++
std::vector<int> v = {1, 2, 3};
v.resize(5);      // {1, 2, 3, 0, 0} — size = 5
v.resize(7, -1);  // {1, 2, 3, 0, 0, -1, -1}
v.resize(2);      // {1, 2} — зайві видалено
```

## Практичне значення

```c++
// Погано: багато перевиділень
std::vector<int> v;
for (int i = 0; i < 100000; ++i)
    v.push_back(i);  // кілька перевиділень

// Добре: резервуємо з самого початку
std::vector<int> v;
v.reserve(100000);
for (int i = 0; i < 100000; ++i)
    v.push_back(i);  // жодного перевиділення
```

## Правило

Якщо ви знаєте або можете оцінити кількість елементів заздалегідь — використовуйте `reserve()`.
