# Урок №102. Параметри і аргументи функцій

Параметри — змінні у визначенні функції, аргументи — фактичні значення при виклику.

## Параметри

```c++
void printSum(int a, int b)  // a, b — параметри
{
    std::cout << a + b;
}
```

## Аргументи

```c++
int main()
{
    printSum(5, 3);  // 5, 3 — аргументи
    return 0;
}
```

## Параметри за замовчуванням

```c++
void greet(std::string name = "гість")
{
    std::cout << "Привіт, " << name << "!";
}

greet();         // "Привіт, гість!"
greet("Анна");   // "Привіт, Анна!"
```

Параметри зі значеннями за замовчуванням мають бути крайніми справа:

```c++
void f(int a, int b = 0, int c = 0);  // OK
// void f(int a = 0, int b);  // ПОМИЛКА
```

## Передача за значенням (копія)

```c++
void inc(int x)
{
    x = x + 1;  // змінюється ТІЛЬКИ копія
}

int a = 5;
inc(a);
std::cout << a;  // 5 (не змінилось)
```

## Передача за посиланням

```c++
void inc(int& x)
{
    x = x + 1;  // змінюється оригінал
}

int a = 5;
inc(a);
std::cout << a;  // 6
```

## Передача const посиланням

```c++
void print(const std::string& s)
{
    std::cout << s;  // не копіює, не змінює
}
```

## Кількість параметрів

C++ не обмежує кількість, але для читабельності — не більше 4-5.

## Правило

Використовуйте:
- `T` для малих типів (int, double, char)
- `const T&` для великих типів (std::string, вектор)
- `T&` коли потрібно змінити аргумент
- `T*` коли аргумент може бути nullptr
