# Урок №118. Лямбда-вирази

Лямбда-вираз (C++11) — це анонімна функція, яку можна визначити прямо в місці використання.

## Синтаксис

```c++
[захоплення](параметри) -> тип_повернення { тіло };
```

## Простий приклад

```c++
auto greet = []() {
    std::cout << "Hello!\n";
};

greet();  // Hello!
```

## З параметрами

```c++
auto add = [](int a, int b) -> int {
    return a + b;
};

std::cout << add(5, 3);  // 8

// Тип повернення можна опустити (auto виведе):
auto multiply = [](int a, int b) { return a * b; };
```

## Лямбди та алгоритми

```c++
#include <algorithm>
#include <vector>

std::vector<int> v = {5, 2, 8, 1, 9};

// Сортування за спаданням
std::sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;
});

// Фільтрація
auto it = std::find_if(v.begin(), v.end(), [](int x) {
    return x > 5;
});
```

## Приклад: for_each

```c++
std::vector<int> v = {1, 2, 3, 4, 5};

std::for_each(v.begin(), v.end(), [](int x) {
    std::cout << x * 2 << " ";
});
// 2 4 6 8 10
```

## Лямбда як callback

```c++
void repeat(int n, std::function<void(int)> action)
{
    for (int i = 0; i < n; ++i)
        action(i);
}

int main()
{
    repeat(5, [](int i) {
        std::cout << "Ітерація " << i << "\n";
    });
    return 0;
}
```

## Зберігання в змінній

```c++
// auto — найпростіше
auto func = [](int x) { return x * x; };
std::cout << func(5);  // 25

// std::function — для передачі
std::function<int(int)> squared = [](int x) { return x * x; };
```

## constexpr лямбди (C++17)

```c++
constexpr auto square = [](int x) { return x * x; };
static_assert(square(5) == 25);  // compile-time
```

## Правило

Лямбди роблять код коротшим і чистішим. Використовуйте їх для коротких одноразових функцій, особливо з алгоритмами STL.
