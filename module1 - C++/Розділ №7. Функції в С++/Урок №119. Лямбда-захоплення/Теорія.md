# Урок №119. Лямбда-захоплення

Лямбди можуть "захоплювати" змінні з навколишньої області видимості.

## Захоплення за значенням [=]

Копіює змінні всередину лямбди:

```c++
int multiplier = 3;
auto times = [=](int x) {    // захопити ВСЕ за значенням
    return x * multiplier;    // multiplier — копія
};

std::cout << times(5);  // 15
```

## Захоплення за посиланням [&]

```c++
int counter = 0;
auto increment = [&]() {   // захопити ВСЕ за посиланням
    ++counter;               // змінює оригінал
};

increment();
std::cout << counter;  // 1
```

## Явне захоплення

```c++
int a = 5, b = 10;

// Конкретні змінні:
auto f1 = [a]() { return a; };           // a за значенням
auto f2 = [&a]() { a++; };              // a за посиланням
auto f3 = [a, &b]() { return a + b; };  // a за значенням, b за посиланням
```

## Поєднання

```c++
int x = 1, y = 2, z = 3;

auto f = [=, &y]() {   // все за значенням, крім y (за посиланням)
    // x = 10;  // ПОМИЛКА! x за значенням
    y = 20;     // OK
    return x + y + z;
};
```

## mutable — зміна захоплених копій

За замовчуванням лямбда не може змінювати захоплені за значенням змінні. `mutable` дозволяє:

```c++
int count = 0;
auto counter = [count]() mutable {
    return ++count;  // змінюємо копію count
};

std::cout << counter();  // 1
std::cout << counter();  // 2
std::cout << count;      // 0 (оригінал не змінився)
```

## Захоплення this

У методах класу:

```c++
class Calculator
{
    int factor = 10;
public:
    auto getMultiplier() const
    {
        return [this](int x) {  // захоплює вказівник на об'єкт
            return x * factor;  // доступ до членів
        };
    }
};
```

## Ініціалізація захоплення (C++14)

```c++
auto lambda = [x = 5](int y) {  // x = 5 — нове ім'я/значення
    return x + y;
};

std::cout << lambda(3);  // 8

// Переміщення унікального вказівника
auto p = std::make_unique<int>(42);
auto f = [ptr = std::move(p)]() {  // переміщення в лямбду
    return *ptr;
};
```

## Життєвий цикл захоплення

```c++
auto createLambda()
{
    int x = 42;
    return [&x]() { return x; };  // НЕБЕЗПЕЧНО! x знищується після виходу
    // return [x]() { return x; };  // БЕЗПЕЧНО: копія
}
```

## Таблиця захоплення

| Синтаксис | Опис |
|-----------|------|
| `[=]` | Все за значенням |
| `[&]` | Все за посиланням |
| `[x]` | x за значенням |
| `[&x]` | x за посиланням |
| `[=, &x]` | Все за значенням, x за посиланням |
| `[&, x]` | Все за посиланням, x за значенням |
| `[this]` | this (доступ до членів класу) |
| `[*this]` | Копія об'єкта (C++17) |

## Правило

Захоплюйте за посиланням, коли потрібно змінити змінну. Захоплюйте за значенням, коли лямбда "живе" довше області видимості.
