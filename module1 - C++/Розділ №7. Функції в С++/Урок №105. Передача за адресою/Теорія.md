# Урок №105. Передача за адресою

Передача за адресою — передача вказівника на об'єкт.

## Синтаксис

```c++
void func(int* ptr)  // ptr — вказівник
{
    if (ptr)
        *ptr = 100;
}
```

## Приклад

```c++
#include <iostream>

void setToZero(int* ptr)
{
    if (ptr)
        *ptr = 0;
}

int main()
{
    int x = 5;
    setToZero(&x);   // передаємо адресу
    std::cout << x;  // 0
    
    setToZero(nullptr);  // безпечно: перевірка всередині
    
    return 0;
}
```

## Посилання vs Вказівник

```c++
void byRef(int& x)   // не може бути nullptr
{
    x = 10;
}

void byPtr(int* x)   // може бути nullptr
{
    if (x)           // потрібна перевірка
        *x = 10;
}
```

## Модифікація вказівника

Через `**` можна змінити сам вказівник:

```c++
void allocate(int** ptr, int size)
{
    *ptr = new int[size];
    for (int i = 0; i < size; ++i)
        (*ptr)[i] = i;
}

int main()
{
    int* arr = nullptr;
    allocate(&arr, 5);
    delete[] arr;
    return 0;
}
```

## Передача масиву

```c++
void print(int* arr, int size)
{
    for (int i = 0; i < size; ++i)
        std::cout << arr[i] << " ";
}

int data[] = {1, 2, 3};
print(data, 3);  // масив "розпадається" на вказівник
```

## const T* — тільки читання

```c++
void readOnly(const int* data, int size)
{
    for (int i = 0; i < size; ++i)
        std::cout << data[i];
    // data[0] = 5;  // ПОМИЛКА!
}
```

## Використання з опціональними даними

```c++
void configure(int* timeout = nullptr)
{
    if (timeout)
        std::cout << "Таймаут: " << *timeout << "\n";
    else
        std::cout << "Таймаут за замовчуванням\n";
}
```

## Коли використовувати?

- Потрібна сумісність з C
- Аргумент може бути nullptr
- Робота з динамічною пам'яттю

## Правило

Віддавайте перевагу посиланням перед вказівниками. Використовуйте вказівники тільки коли аргумент може бути nullptr.
