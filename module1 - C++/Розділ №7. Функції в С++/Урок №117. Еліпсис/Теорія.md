# Урок №117. Еліпсис

Еліпсис (`...`) дозволяє функції приймати змінну кількість аргументів.

## C-style варіативні функції

```c++
#include <cstdarg>

int sum(int count, ...)
{
    int total = 0;
    va_list args;
    va_start(args, count);
    
    for (int i = 0; i < count; ++i)
        total += va_arg(args, int);
    
    va_end(args);
    return total;
}

int main()
{
    std::cout << sum(3, 10, 20, 30);  // 60
    std::cout << sum(5, 1, 2, 3, 4, 5);  // 15
    return 0;
}
```

## Небезпека

Еліпсис НЕ перевіряє типи!

```c++
sum(2, 10, 20.5);  // НЕБЕЗПЕЧНО! double прочитається як ...?
// Без перевірки типів — невизначена поведінка
```

## Сучасна альтернатива: variadic templates (C++11)

```c++
template<typename... Args>
auto sum(Args... args)
{
    return (args + ...);  // fold expression (C++17)
}

int main()
{
    std::cout << sum(1, 2, 3);         // 6
    std::cout << sum(1, 2, 3, 4, 5);   // 15
    return 0;
}
```

## Variadic templates з рекурсією

```c++
template<typename T>
T sum(T t) { return t; }  // базовий випадок

template<typename T, typename... Args>
T sum(T first, Args... rest)
{
    return first + sum(rest...);  // рекурсивний виклик
}
```

## Приклад: printf

Найвідоміша варіативна функція — `printf`:

```c++
#include <cstdio>

printf("Число: %d\n", 42);
printf("Ім'я: %s, вік: %d\n", "Анна", 25);
```

## Коли використовувати?

- C-style еліпсис — тільки для сумісності з C
- Variadic templates — для типобезпечних варіативних функцій
- `std::initializer_list` — для однотипних аргументів

```c++
#include <initializer_list>

int sum(std::initializer_list<int> list)
{
    int total = 0;
    for (int x : list) total += x;
    return total;
}

int main()
{
    std::cout << sum({1, 2, 3, 4, 5});  // 15
    return 0;
}
```

## Правило

Уникайте C-style еліпсису в сучасному C++. Використовуйте variadic templates або `std::initializer_list`.
