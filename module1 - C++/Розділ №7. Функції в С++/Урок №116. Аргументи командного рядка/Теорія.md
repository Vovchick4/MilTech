# Урок №116. Аргументи командного рядка

Програми можуть отримувати аргументи через командний рядок.

## main() з параметрами

```c++
int main(int argc, char* argv[])
{
    // argc — кількість аргументів
    // argv — масив C-style рядків
    return 0;
}
```

## Приклад

```c++
#include <iostream>

int main(int argc, char* argv[])
{
    std::cout << "Кількість аргументів: " << argc << "\n";
    
    for (int i = 0; i < argc; ++i)
    {
        std::cout << "Аргумент " << i << ": " << argv[i] << "\n";
    }
    
    return 0;
}
```

Запуск: `program.exe hello world 42`
```
Кількість аргументів: 4
Аргумент 0: program.exe
Аргумент 1: hello
Аргумент 2: world
Аргумент 3: 42
```

## Перший аргумент — ім'я програми

`argv[0]` завжди містить ім'я/шлях до виконуваного файлу.

## Конвертація аргументів

Аргументи передаються як рядки. Для чисел потрібна конвертація:

```c++
#include <iostream>
#include <cstdlib>

int main(int argc, char* argv[])
{
    if (argc < 3)
    {
        std::cerr << "Використання: program.exe число1 число2\n";
        return 1;
    }
    
    int a = std::atoi(argv[1]);
    int b = std::atoi(argv[2]);
    
    std::cout << a << " + " << b << " = " << a + b << "\n";
    return 0;
}
```

## Кращий парсинг: std::from_chars (C++17)

```c++
#include <iostream>
#include <charconv>

int main(int argc, char* argv[])
{
    if (argc < 3) return 1;
    
    int a, b;
    std::from_chars(argv[1], argv[1] + strlen(argv[1]), a);
    std::from_chars(argv[2], argv[2] + strlen(argv[2]), b);
    
    std::cout << a + b << "\n";
    return 0;
}
```

## Прапорці (flags)

```c++
int main(int argc, char* argv[])
{
    bool verbose = false;
    std::string filename = "default.txt";
    
    for (int i = 1; i < argc; ++i)
    {
        std::string arg = argv[i];
        if (arg == "-v" || arg == "--verbose")
            verbose = true;
        else if (arg == "-o" && i + 1 < argc)
            filename = argv[++i];
    }
    
    if (verbose)
        std::cout << "Файл: " << filename << "\n";
    
    return 0;
}
```

## Запуск з командного рядка

```cmd
myprogram.exe --input data.txt --output result.txt

// PowerShell:
./myprogram --input data.txt --output result.txt
```

## Правило

Використовуйте `argc/argv` для простих програм. Для складних CLI-інтерфейсів використовуйте бібліотеки (CLI11, cxxopts).
