# Урок №67. Оператори умовного розгалуження if/else

`if` дозволяє виконати код тільки якщо умова істинна.

## Базовий if

```c++
if (умова)
{
    // код виконується, якщо умова == true
}
```

```c++
int x = 5;
if (x > 0)
{
    std::cout << "x додатне\n";
}
```

## if-else

```c++
if (умова)
{
    // код, якщо true
}
else
{
    // код, якщо false
}
```

```c++
int x = -3;
if (x >= 0)
{
    std::cout << "Невід'ємне\n";
}
else
{
    std::cout << "Від'ємне\n";
}
```

## if-else if-else

```c++
int score = 85;

if (score >= 90)
{
    std::cout << "Відмінно\n";
}
else if (score >= 70)
{
    std::cout << "Добре\n";
}
else if (score >= 50)
{
    std::cout << "Задовільно\n";
}
else
{
    std::cout << "Незадовільно\n";
}
```

## Вкладені if

```c++
int x = 10, y = 20;

if (x > 0)
{
    if (y > 0)
    {
        std::cout << "Обидва додатні\n";
    }
    else
    {
        std::cout << "Тільки x додатне\n";
    }
}
```

## Стиль: фігурні дужки

```c++
// Без дужок — тільки перший рядок належить if:
if (x > 0)
    std::cout << "Додатне\n";  // до if
    std::cout << "Завжди\n";   // НЕ до if!

// З дужками — чітко:
if (x > 0)
{
    std::cout << "Додатне\n";
    std::cout << "Це теж до if\n";
}
```

## Поширені помилки

```c++
// Помилка: = замість ==
if (x = 5)  // присвоювання! завжди true
{
    // виконується завжди
}

// Помилка: крапка з комою після умови
if (x > 0);  // ← крапка з комою! if нічого не робить
{
    std::cout << "Це виконується завжди\n";
}
```

## Тернарний оператор як альтернатива

```c++
// if-else:
int max;
if (a > b)
    max = a;
else
    max = b;

// Тернарний:
int max = (a > b) ? a : b;
```

## Приклад

```c++
#include <iostream>

int main()
{
    std::cout << "Введіть число: ";
    int x;
    std::cin >> x;
    
    if (x % 2 == 0)
        std::cout << x << " — парне\n";
    else
        std::cout << x << " — непарне\n";
    
    return 0;
}
```

## Правило

Завжди використовуйте фігурні дужки `{}` для if/else, навіть для одного рядка.
