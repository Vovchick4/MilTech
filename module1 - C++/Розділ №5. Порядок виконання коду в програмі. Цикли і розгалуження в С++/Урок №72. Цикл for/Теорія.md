# Урок №72. Цикл for

`for` — це цикл, який об'єднує ініціалізацію, умову та зміну лічильника в одному рядку.

## Синтаксис

```c++
for (ініціалізація; умова; зміна)
{
    // тіло циклу
}
```

## Базовий приклад

```c++
for (int i = 0; i < 5; ++i)
{
    std::cout << i << " ";
}
// Виведе: 0 1 2 3 4
```

## Як це працює?

```
1. Ініціалізація: i = 0 (виконується один раз на початку)
2. Умова: i < 5? Якщо true — виконати тіло
3. Тіло: std::cout << i
4. Зміна: ++i
5. Повернутися до кроку 2
```

## Кілька змінних

```c++
for (int i = 0, j = 10; i < j; ++i, --j)
{
    std::cout << "i=" << i << " j=" << j << "\n";
}
```

## Частини циклу for можуть бути пропущені

```c++
// Без ініціалізації
int i = 0;
for (; i < 5; ++i) { }

// Без умови (нескінченний)
for (int i = 0; ; ++i)
{
    if (i > 10) break;
}

// Без зміни
for (int i = 0; i < 5; )
{
    ++i;  // зміна всередині
}

// Пустий цикл (всі частини)
for (;;) { }  // нескінченний цикл
```

## Типові використання

### Обхід масиву

```c++
int arr[] = {10, 20, 30, 40, 50};

for (int i = 0; i < 5; ++i)
{
    std::cout << arr[i] << " ";
}
```

### Сума чисел

```c++
int sum = 0;
for (int i = 1; i <= 100; ++i)
{
    sum += i;
}
// sum = 5050
```

### Таблиця множення

```c++
for (int i = 1; i <= 10; ++i)
{
    for (int j = 1; j <= 10; ++j)
    {
        std::cout << i * j << "\t";
    }
    std::cout << "\n";
}
```

## for з плаваючою крапкою

```c++
for (double x = 0.0; x <= 1.0; x += 0.25)
{
    std::cout << x << " ";  // 0 0.25 0.5 0.75 1
}
```

## Вкладені цикли for

```c++
for (int row = 0; row < 3; ++row)
{
    for (int col = 0; col < 5; ++col)
    {
        std::cout << "(" << row << "," << col << ") ";
    }
    std::cout << "\n";
}
```

## for vs while

```c++
// for — коли кількість ітерацій ВІДОМА
for (int i = 0; i < n; ++i) { process(arr[i]); }

// while — коли кількість НЕВІДОМА
while (std::cin >> x) { process(x); }
```

## Правило

Використовуйте `for`, коли потрібен лічильник з відомою кількістю ітерацій. Використовуйте `while`, коли умова залежить від зовнішніх даних.
