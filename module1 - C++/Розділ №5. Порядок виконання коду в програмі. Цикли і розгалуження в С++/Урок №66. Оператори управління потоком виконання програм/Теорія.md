# Урок №66. Оператори управління потоком виконання програм

## Потік виконання

За замовчуванням програма виконується зверху вниз, рядок за рядком. Оператори управління потоком змінюють цей порядок.

## Види потоків управління

### 1. Послідовний — рядок за рядком

```c++
int x = 5;
int y = 10;
int z = x + y;  // виконується по порядку
```

### 2. Розгалуження — вибір одного з варіантів

```c++
if (x > 0)
    std::cout << "Додатне\n";
else
    std::cout << "Від'ємне\n";
```

### 3. Цикли — повторення блоку коду

```c++
for (int i = 0; i < 10; ++i)
    std::cout << i << " ";
```

### 4. Стрибки — перехід в інше місце коду

```c++
goto label;  // не рекомендується!
label:
    // ...
```

## Оператори управління

| Категорія | Оператори |
|-----------|-----------|
| Умовні | `if`, `else`, `switch`, `case`, `default` |
| Цикли | `while`, `do-while`, `for`, `range-based for` |
| Стрибки | `break`, `continue`, `goto`, `return` |
| Винятки | `try`, `catch`, `throw` |

## Важливі концепції

### Структурне програмування

Сучасні програми пишуться за принципами структурного програмування:
- Уникайте `goto`
- Використовуйте блоки `{}`
- Одна точка входу, одна точка виходу для функцій

### Правило

Кожен оператор управління повинен мати зрозумілу умову. Використовуйте дужки `{}` навіть для одного рядка.
