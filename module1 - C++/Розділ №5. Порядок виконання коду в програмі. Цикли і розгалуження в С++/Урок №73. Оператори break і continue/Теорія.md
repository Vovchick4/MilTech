# Урок №73. Оператори break і continue

`break` і `continue` — оператори управління циклом.

## break

`break` негайно завершує цикл або `switch`:

```c++
for (int i = 0; i < 10; ++i)
{
    if (i == 5)
        break;  // цикл завершується
    
    std::cout << i << " ";
}
// Виведе: 0 1 2 3 4
```

### break у while

```c++
int x;
while (true)
{
    std::cout << "Введіть число (0 для виходу): ";
    std::cin >> x;
    
    if (x == 0)
        break;  // вихід з циклу
    
    std::cout << "Ви ввели: " << x << "\n";
}
```

### break у switch

```c++
switch (value)
{
    case 1:
        // код
        break;  // вихід з switch
    case 2:
        // код
        break;
}
```

## continue

`continue` пропускає решту тіла циклу і переходить до наступної ітерації:

```c++
for (int i = 0; i < 10; ++i)
{
    if (i % 2 == 0)
        continue;  // пропустити парні
    
    std::cout << i << " ";
}
// Виведе: 1 3 5 7 9
```

### continue у while (обережно!)

```c++
int i = 0;
while (i < 10)
{
    if (i % 2 == 0)
        continue;  // НЕБЕЗПЕЧНО! i не збільшується — нескінченний цикл!
    std::cout << i;
    ++i;
}

// Правильно:
int i = 0;
while (i < 10)
{
    ++i;  // збільшуємо ДО continue
    if (i % 2 == 0)
        continue;
    std::cout << i;
}
```

## Практичний приклад

```c++
#include <iostream>

int main()
{
    int sum = 0;
    int count = 0;
    
    std::cout << "Введіть числа (від'ємне для завершення):\n";
    
    while (true)
    {
        int x;
        std::cin >> x;
        
        if (x < 0)
            break;  // вихід з циклу
        
        if (x == 0)
            continue;  // пропустити 0
        
        sum += x;
        ++count;
    }
    
    std::cout << "Сума: " << sum << "\n";
    std::cout << "Кількість: " << count << "\n";
    
    return 0;
}
```

## break і вкладені цикли

`break` завершує ТІЛЬКИ внутрішній цикл:

```c++
for (int i = 0; i < 3; ++i)
{
    for (int j = 0; j < 3; ++j)
    {
        if (j == 1)
            break;  // вихід тільки з внутрішнього циклу
        std::cout << i << j << " ";
    }
    std::cout << "\n";
}
// 00 10 20
```

## Порівняння

| Оператор | Дія |
|----------|-----|
| `break` | Завершує цикл повністю |
| `continue` | Пропускає поточну ітерацію |
| `return` | Завершує функцію (і цикл) |

## Правило

Використовуйте `break` для дострокового виходу з циклу при знайденому результаті. Використовуйте `continue` щоб уникнути глибокої вкладеності if в циклі.
