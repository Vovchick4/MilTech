# Урок №71. Цикл do while

`do-while` — це цикл з післяумовою. Тіло циклу виконується хоча б один раз, потім перевіряється умова.

## Синтаксис

```c++
do
{
    // тіло циклу — виконується хоча б один раз
}
while (умова);  // ← крапка з комою!
```

## Простий приклад

```c++
int count = 0;

do
{
    std::cout << count << " ";
    ++count;
}
while (count < 5);
// Виведе: 0 1 2 3 4
```

## while vs do-while

```c++
// while — умова спочатку, тіло може не виконатися
int x = 10;
while (x < 5)
{
    std::cout << "Не виконається\n";
}

// do-while — тіло виконується, потім умова
int y = 10;
do
{
    std::cout << "Виконається один раз\n";
}
while (y < 5);
```

## Коли використовувати?

Найкращий приклад — меню/ввід даних, де потрібно запитати користувача хоча б один раз:

```c++
#include <iostream>

int main()
{
    int choice;
    
    do
    {
        std::cout << "\nМеню:\n";
        std::cout << "1. Додати\n";
        std::cout << "2. Видалити\n";
        std::cout << "3. Вийти\n";
        std::cout << "Вибір: ";
        std::cin >> choice;
        
        switch (choice)
        {
            case 1: std::cout << "Додаємо...\n"; break;
            case 2: std::cout << "Видаляємо...\n"; break;
            case 3: std::cout << "До побачення!\n"; break;
            default: std::cout << "Невірний вибір\n";
        }
    }
    while (choice != 3);
    
    return 0;
}
```

## Введення з перевіркою

```c++
int value;
do
{
    std::cout << "Введіть число від 1 до 10: ";
    std::cin >> value;
}
while (value < 1 || value > 10);

std::cout << "Ви ввели: " << value;
```

## Пошук числа

```c++
#include <iostream>
#include <cstdlib>
#include <ctime>

int main()
{
    std::srand(std::time(0));
    int secret = std::rand() % 100 + 1;
    int guess;
    
    do
    {
        std::cout << "Вгадайте число (1-100): ";
        std::cin >> guess;
        
        if (guess < secret)
            std::cout << "Більше!\n";
        else if (guess > secret)
            std::cout << "Менше!\n";
    }
    while (guess != secret);
    
    std::cout << "Вгадали!\n";
    return 0;
}
```

## Важливо: крапка з комою після while

```c++
do
{
    // тіло
}
while (умова);  // ← ОБОВ'ЯЗКОВО крапка з комою!
```

## Таблиця порівняння

| Характеристика | while | do-while |
|---------------|-------|----------|
| Перевірка умови | Перед тілом | Після тіла |
| Мінімум виконань | 0 | 1 |
| Типове використання | Коли може не виконуватися | Коли має виконатися хоча б раз |

## Правило

Використовуйте `do-while`, коли код має виконатися хоча б один раз незалежно від умови (наприклад, меню, запит вводу).
