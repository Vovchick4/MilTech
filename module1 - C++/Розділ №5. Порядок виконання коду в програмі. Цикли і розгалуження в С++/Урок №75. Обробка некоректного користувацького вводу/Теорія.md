# Урок №75. Обробка некоректного користувацького вводу

Користувачі можуть ввести некоректні дані. Програма повинна це обробляти.

## Проблема: неправильний тип даних

```c++
int age;
std::cout << "Введіть вік: ";
std::cin >> age;  // користувач ввів "abc"
// std::cin переходить у стан помилки, age = 0 або не змінюється
```

## Стани потоку cin

| Стан | Опис |
|------|------|
| `good()` | Все добре |
| `fail()` | Помилка формату (введено не число) |
| `bad()` | Фатальна помилка |
| `eof()` | Кінець файлу |

## Базова обробка

```c++
#include <iostream>
#include <limits>

int main()
{
    int x;
    
    std::cout << "Введіть число: ";
    std::cin >> x;
    
    if (std::cin.fail())
    {
        std::cout << "Помилка вводу!\n";
        std::cin.clear();                // скидаємо стан помилки
        std::cin.ignore(32767, '\n');     // очищуємо буфер
    }
    else
    {
        std::cout << "Ви ввели: " << x << "\n";
    }
    
    return 0;
}
```

## Функція безпечного вводу

```c++
#include <iostream>
#include <limits>

int getInt()
{
    while (true)
    {
        std::cout << "Введіть ціле число: ";
        int x;
        std::cin >> x;
        
        if (std::cin.fail())
        {
            std::cin.clear();
            std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
            std::cout << "Некоректний ввід. Спробуйте ще.\n";
        }
        else
        {
            std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
            return x;
        }
    }
}
```

## Ввід з діапазоном

```c++
int getIntBetween(int min, int max)
{
    while (true)
    {
        int x = getInt();
        
        if (x >= min && x <= max)
            return x;
        
        std::cout << "Число має бути від " << min << " до " << max << "\n";
    }
}

// Використання:
int age = getIntBetween(0, 150);
```

## Ввід рядка з перевіркою

```c++
#include <iostream>
#include <string>
#include <limits>

std::string getString()
{
    std::string s;
    std::getline(std::cin, s);
    
    if (s.empty())
    {
        std::cout << "Рядок не може бути порожнім!\n";
        return getString();
    }
    
    return s;
}
```

## Повний приклад

```c++
#include <iostream>
#include <limits>

int getInt()
{
    while (true)
    {
        int x;
        std::cin >> x;
        
        if (std::cin.fail())
        {
            std::cin.clear();
            std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
            std::cout << "Некоректно! Спробуйте ще: ";
        }
        else
        {
            std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
            return x;
        }
    }
}

int main()
{
    std::cout << "Введіть вік: ";
    int age = getInt();
    std::cout << "Ваш вік: " << age << "\n";
    return 0;
}
```

## Важливі моменти

```c++
// Після std::cin >> x, якщо далі потрібен getline:
std::cin >> x;
std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');  // очистити \n
std::getline(std::cin, line);

// Або просто:
std::cin >> x;
std::cin >> std::ws;  // пропустити пробільні символи
std::getline(std::cin, line);
```

## Правило

Завжди перевіряйте стан `std::cin` після вводу. Очищуйте буфер після помилки. Ніколи не довіряйте користувацькому вводу.

## Тест

1. Що станеться, якщо ввести "abc" замість числа?
2. Які три кроки потрібні після помилки вводу?
3. Чому потрібно очищувати буфер після `std::cin >> x` перед `getline`?
