# Урок №68. Оператор switch

`switch` — це оператор вибору, який дозволяє порівняти значення з набором констант.

## Синтаксис

```c++
switch (вираз)
{
    case константа1:
        // код
        break;
    case константа2:
        // код
        break;
    default:
        // код (якщо жоден case не підійшов)
        break;
}
```

## Приклад

```c++
int digit;
std::cin >> digit;

switch (digit)
{
    case 1:
        std::cout << "Один\n";
        break;
    case 2:
        std::cout << "Два\n";
        break;
    case 3:
        std::cout << "Три\n";
        break;
    default:
        std::cout << "Невідомо\n";
        break;
}
```

## Еквівалентний if/else

```c++
if (digit == 1)
    std::cout << "Один\n";
else if (digit == 2)
    std::cout << "Два\n";
else if (digit == 3)
    std::cout << "Три\n";
else
    std::cout << "Невідомо\n";
```

`switch` часто читабельніший за довгі ланцюжки if-else if.

## break — вихід з switch

Без `break` виконання "провалюється" в наступний case (fall-through):

```c++
int x = 2;
switch (x)
{
    case 1:
        std::cout << "1 ";
    case 2:
        std::cout << "2 ";   // виконається
    case 3:
        std::cout << "3 ";   // виконається (провалювання!)
        break;
}
// Виведе "2 3 "
```

## Навмисне провалювання

Інколи fall-through корисний:

```c++
char grade = 'B';

switch (grade)
{
    case 'A':
    case 'B':
    case 'C':
        std::cout << "Склав\n";
        break;
    case 'D':
    case 'E':
        std::cout.WriteLine("Не склав\n");
        break;
    default:
        std::cout.WriteLine("Невірна оцінка\n");
        break;
}
```

## Що можна використовувати в case?

Тільки цілочисельні константи та enum:

```c++
constexpr int OPTION_A = 1;

switch (opt)
{
    case 1:        // OK
    case 'A':      // OK (char — цілий тип)
    case OPTION_A: // OK (constexpr)
    // case x:     // ПОМИЛКА! x — змінна (не константа)
}
```

## switch з enum

```c++
enum class Color { RED, GREEN, BLUE };

Color c { Color::RED };

switch (c)
{
    case Color::RED:
        std::cout << "Червоний\n";
        break;
    case Color::GREEN:
        std::cout << "Зелений\n";
        break;
    case Color::BLUE:
        std::cout << "Синій\n";
        break;
}
```

## Коли використовувати switch?

- Коли є 3+ варіанти для однієї змінної
- Коли варіанти — константи
- Для enum типів

## Правило

Завжди додавайте `break` у кожен case (якщо не потрібне провалювання). Додавайте `default` для обробки несподіваних значень.
