# Урок №84. Вказівники

`Вказівник` (pointer) — це змінна, яка зберігає адресу пам'яті іншої змінної.

## Адреса в пам'яті

Кожна змінна має адресу. Оператор `&` повертає адресу:

```c++
int x = 5;
std::cout << &x;  // щось на кшталт 0x7ffdf5e0 (адреса x)
```

## Оголошення вказівника

```c++
int x = 5;
int* ptr = &x;  // ptr — вказівник на int, зберігає адресу x
```

## Розіменування (*)

Оператор `*` (розіменування) — отримує значення за адресою:

```c++
int x = 5;
int* ptr = &x;

std::cout << ptr;   // адреса
std::cout << *ptr;  // 5 (значення x через вказівник)

*ptr = 10;          // зміна x через вказівник
std::cout << x;     // 10
```

## Візуалізація

```
Пам'ять:
x = 5   (адреса 0x100)
ptr = 0x100 (зберігає адресу x)

*ptr → звертаємось за адресою 0x100 → отримуємо 5
```

## Типи вказівників

```c++
int* p1;        // вказівник на int
double* p2;     // вказівник на double
char* p3;       // вказівник на char

int x = 5;
double y = 3.14;
int* ip = &x;   // OK
// int* ip = &y;  // ПОМИЛКА! тип не співпадає
```

## Вказівники та функції

```c++
void setToZero(int* ptr)
{
    *ptr = 0;  // змінюємо значення за адресою
}

int main()
{
    int x = 5;
    setToZero(&x);  // передаємо адресу
    std::cout << x;  // 0
    return 0;
}
```

## Розмір вказівника

```c++
std::cout << sizeof(int*);   // 8 (на 64-бітній системі)
std::cout << sizeof(double*); // 8
std::cout << sizeof(char*);   // 8
// Розмір ВСІХ вказівників однаковий — адреса пам'яті
```

## Поширені помилки

```c++
int* ptr;       // неініціалізований (містить сміття)
*ptr = 5;       // НЕБЕЗПЕЧНО! запис за випадковою адресою

int x = 5;
int* p = &x;    // OK
int* q = p;     // OK — копіювання адреси
*q = 10;        // змінює x
```

## Правило

Завжди ініціалізуйте вказівники. Перед розіменуванням перевіряйте, чи вказівник не нульовий.
