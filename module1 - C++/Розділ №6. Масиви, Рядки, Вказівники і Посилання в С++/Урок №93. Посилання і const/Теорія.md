# Урок №93. Посилання і const

Поєднання посилань та `const` — потужний інструмент.

## const T& для параметрів

Найпоширеніше використання — передача великих об'єктів без копіювання, з гарантією незмінності:

```c++
void process(const std::string& s)
{
    // s — посилання (без копіювання)
    // const — не можна змінити
    std::cout << s.length();
}
```

## const T& для повернення

```c++
class Container
{
    int data[100];
public:
    const int& get(int index) const
    {
        return data[index];  // const посилання на внутрішні дані
    }
};
```

## const посилання продовжує життя тимчасового об'єкта

```c++
std::string createString() { return "Hello"; }

int main()
{
    const std::string& ref = createString();
    // ref — посилання на тимчасовий об'єкт
    // const подовжує життя тимчасового об'єкта до кінця області видимості ref
    std::cout << ref;  // OK
    return 0;
}
```

## const T& та літерали

```c++
void foo(const int& x) { std::cout << x; }

foo(42);  // OK: const int& може прийняти літерал
```

## Звичайне посилання vs const посилання

```c++
int x = 5;

int& r1 = x;      // можна змінити x через r1
const int& r2 = x; // не можна змінити x через r2

r1 = 10;   // OK
// r2 = 10;  // ПОМИЛКА
x = 10;     // OK (зміна через оригінал)
```

## Правило "const зліва"

Правило читання оголошень: читайте справа наліво.

```c++
const int& ref;  // ref — посилання на const int
int const& ref;  // те саме
```

## Приклад: пошук у контейнері

```c++
#include <iostream>
#include <vector>

const int& findMax(const std::vector<int>& vec)
{
    int maxIdx = 0;
    for (int i = 1; i < vec.size(); ++i)
        if (vec[i] > vec[maxIdx]) maxIdx = i;
    return vec[maxIdx];  // const посилання
}

int main()
{
    std::vector<int> data = {4, 7, 1, 9, 3};
    const int& max = findMax(data);
    std::cout << max;  // 9
    return 0;
}
```

## Поширена помилка: посилання на локальну змінну

```c++
int& getValue()  // НЕБЕЗПЕЧНО!
{
    int x = 5;
    return x;  // x знищується після виходу!
}

int main()
{
    int& ref = getValue();
    std::cout << ref;  // НЕВИЗНАЧЕНО! x вже не існує
    return 0;
}
```

## Правило

Завжди використовуйте `const T&` для параметрів, які не змінюються. Ніколи не повертайте посилання на локальну змінну.
