# Урок №82. Рядки C-style

C-style рядки — це масиви символів `char[]`, що закінчуються нульовим символом `'\0'`.

## Створення

```c++
char str[] = "Hello";        // {'H','e','l','l','o','\0'} — 6 байтів
char str2[6] = "Hello";      // явний розмір
const char* str3 = "Hello";  // вказівник на константний рядок
```

## Нульовий термінатор '\0'

Кожен C-style рядок обов'язково закінчується `'\0'`. Це ознака кінця рядка:

```c++
char str[3] = "Hi";  // {'H', 'i', '\0'}
```

## Функції для роботи (з <cstring>)

```c++
#include <cstring>

char str1[] = "Hello";
char str2[] = "World";

int len = strlen(str1);              // 5 (без \0)
strcpy(dest, src);                    // копіювання
strcat(str1, str2);                   // конкатенація (увага: розмір!)
int cmp = strcmp(str1, str2);         // порівняння
```

## Введення

```c++
char name[50];
std::cin >> name;         // до першого пробілу
std::cin.getline(name, 50);  // вся строка (безпечно!)
```

## Небезпеки

```c++
char str[5];
strcpy(str, "HelloWorld");  // НЕБЕЗПЕЧНО! переповнення буфера
str[10] = '!';              // вихід за межі
```

## Чому краще std::string?

```c++
// C-style — небезпечно
char s1[20];
strcpy(s1, "Hello");
strcat(s1, " World");  // потрібно стежити за розміром

// std::string — безпечно
std::string s = "Hello";
s += " World";  // автоматичне керування пам'яттю
```

## Правило

У сучасному C++ використовуйте `std::string`, а не C-style рядки. C-style рядки використовуйте тільки при роботі з C-бібліотеками.
