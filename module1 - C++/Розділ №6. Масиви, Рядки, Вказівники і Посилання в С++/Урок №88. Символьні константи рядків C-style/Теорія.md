# Урок №88. Символьні константи рядків C-style

## Рядковий літерал

Рядковий літерал — це послідовність символів у подвійних лапках:

```c++
"Hello, world!"
```

Тип рядкового літерала — `const char[]`.

## Зберігання рядкових літералів

```c++
const char* str = "Hello";
// str — вказівник на перший символ 'H'
// Рядок зберігається в статичній пам'яті
```

## Рядкові літерали + вказівники

```c++
const char* name = "Alex";
std::cout << name;      // "Alex"
std::cout << name[0];   // 'A'
std::cout << *(name + 1);  // 'l'

// name — змінна, може вказувати на інший рядок:
name = "John";
```

## Не можна змінювати рядковий літерал

```c++
char* str = "Hello";
str[0] = 'h';  // НЕВИЗНАЧЕНА ПОВЕДІНКА! (запис у константну пам'ять)
```

Правильно:

```c++
char str[] = "Hello";  // копіює рядок у масив
str[0] = 'h';          // OK: змінюємо копію
```

## Символьні константи

```c++
// Різниця між символом і рядком:
char c = 'A';   // один символ (char)
const char* s = "A";  // рядок з двох символів: 'A' + '\0'

sizeof('A');    // 1
sizeof("A");    // 2 (символ + \0)
```

## Рядкові літерали об'єднуються

```c++
// Сусідні рядкові літерали об'єднуються компілятором:
const char* msg = "Hello " "World";
// "Hello World"
```

## Керуючі символи

```c++
const char* path = "C:\\Users\\Name";     // зворотні слеші
const char* text = "Рядок1\nРядок2";      // новий рядок
const char* tab = "Кол1\tКол2";            // табуляція
```

## Багаторядкові рядки

```c++
const char* poem = "Lorem ipsum dolor sit amet,\n"
                   "consectetur adipiscing elit,\n"
                   "sed do eiusmod tempor incididunt.";
```

## Сирі рядкові літерали (C++11)

```c++
const char* raw = R"(C:\Users\Name\file.txt)";
// Еквівалентно: "C:\\Users\\Name\\file.txt"

const char* raw2 = R"(
    <html>
        <body>
            <p>Hello</p>
        </body>
    </html>
)";
```

## Правило

Використовуйте `std::string` для змінних рядків. Використовуйте `const char*` або `std::string_view` для константних рядків.
