# Урок №98. Введення в std::array

`std::array` (C++11) — це обгортка для фіксованого масиву з інтерфейсом STL.

## Підключення

```c++
#include <array>
```

## Оголошення

```c++
std::array<int, 5> arr;              // неініціалізований
std::array<int, 5> arr = {1, 2, 3};  // {1, 2, 3, 0, 0}
std::array<int, 5> arr {};           // {0, 0, 0, 0, 0}
```

## Доступ до елементів

```c++
std::array<int, 5> arr = {10, 20, 30, 40, 50};

arr[0] = 15;         // без перевірки (як звичайний масив)
arr.at(1) = 25;      // з перевіркою меж (кидає виняток)
arr.front();         // перший (10)
arr.back();          // останній (50)
```

## Розмір та ітерація

```c++
std::array<int, 5> arr = {1, 2, 3, 4, 5};

// Розмір (завжди константа)
int size = arr.size();  // 5

// Range-based for
for (int x : arr)
    std::cout << x << " ";

// З індексом
for (size_t i = 0; i < arr.size(); ++i)
    std::cout << arr[i] << " ";
```

## Переваги над звичайним масивом

```c++
// Звичайний масив
int old[5] = {1, 2, 3, 4, 5};
// int size = sizeof(old) / sizeof(old[0]);  // незручно
// Не можна копіювати: int copy[5] = old;  // ПОМИЛКА!

// std::array
std::array<int, 5> arr = {1, 2, 3, 4, 5};
int size = arr.size();  // зручно
std::array<int, 5> copy = arr;  // копіюється!
```

## Сортування та алгоритми

```c++
#include <algorithm>

std::array<int, 5> arr = {5, 2, 4, 1, 3};
std::sort(arr.begin(), arr.end());  // {1, 2, 3, 4, 5}
std::reverse(arr.begin(), arr.end());  // {5, 4, 3, 2, 1}
```

## Функції з std::array

```c++
#include <array>
#include <iostream>

void print(const std::array<int, 5>& arr)
{
    for (int x : arr)
        std::cout << x << " ";
}

int main()
{
    std::array<int, 5> data = {1, 2, 3, 4, 5};
    print(data);
    return 0;
}
```

## Недолік: фіксований розмір у шаблоні

```c++
// Треба вказувати розмір у шаблоні:
void print(const std::array<int, 5>& arr);  // тільки для масивів з 5!

// Рішення: шаблонна функція
template<size_t N>
void print(const std::array<int, N>& arr) { ... }
```

## std::array vs std::vector

| Характеристика | std::array | std::vector |
|--------------|------------|-------------|
| Розмір | Фіксований (compile-time) | Динамічний |
| Пам'ять | Стек | Купа |
| Продуктивність | Вища | Трохи нижча |
| Копіювання | Просте | Просте |

## Правило

Використовуйте `std::array` замість звичайних масивів C-style, якщо розмір відомий на етапі компіляції.
