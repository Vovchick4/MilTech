# Урок №95. Цикл foreach

Range-based for loop (C++11) — зручний спосіб обходу контейнерів.

## Синтаксис

```c++
for (тип елемент : контейнер)
{
    // тіло циклу
}
```

## Базовий приклад

```c++
int arr[] = {1, 2, 3, 4, 5};

for (int x : arr)
{
    std::cout << x << " ";
}
// 1 2 3 4 5
```

## З auto

```c++
std::vector<std::string> names = {"Анна", "Іван", "Марія"};

for (auto name : names)
{
    std::cout << name << " ";
}
```

## За посиланням (зміна елементів)

```c++
int arr[] = {1, 2, 3, 4, 5};

for (int& x : arr)
{
    x *= 2;  // змінюємо елементи
}
// arr = {2, 4, 6, 8, 10}
```

## Const посилання (ефективно, без копіювання)

```c++
std::vector<std::string> data = {"Hello", "World", "C++"};

for (const auto& s : data)
{
    std::cout << s << " ";  // без копіювання рядків
}
```

## Зі std::map

```c++
std::map<std::string, int> scores = {
    {"Анна", 95}, {"Іван", 87}, {"Марія", 92}
};

for (const auto& pair : scores)
{
    std::cout << pair.first << ": " << pair.second << "\n";
}

// C++17: structured bindings
for (const auto& [name, score] : scores)
{
    std::cout << name << ": " << score << "\n";
}
```

## Порівняння зі звичайним for

```c++
// Звичайний for
for (int i = 0; i < 5; ++i)
    std::cout << arr[i];

// foreach — простіше та безпечніше
for (int x : arr)
    std::cout << x;
```

## Коли foreach НЕ підходить

- Коли потрібен індекс елемента
- Коли потрібно змінювати контейнер під час ітерації
- Коли потрібен зворотний обхід

## Правило

Використовуйте range-based for за замовчуванням для обходу контейнерів. Використовуйте звичайний for тільки коли потрібен індекс.
