# Урок №100. Введення в ітератори

`Ітератор` — це об'єкт, який дозволяє обходити елементи контейнера без знання його внутрішньої структури.

## Навіщо?

```c++
// Індекси працюють тільки для масивів/векторів:
std::vector<int> v = {1, 2, 3};
for (size_t i = 0; i < v.size(); ++i)
    std::cout << v[i];

// Ітератори працюють з БУДЬ-ЯКИМ контейнером:
std::vector<int> v = {1, 2, 3};
for (auto it = v.begin(); it != v.end(); ++it)
    std::cout << *it;
```

## Основні ітератори

```c++
std::vector<int> v = {10, 20, 30, 40, 50};

auto begin = v.begin();  // ітератор на перший елемент
auto end = v.end();      // ітератор на "після останнього"

// Обхід:
for (auto it = begin; it != end; ++it)
    std::cout << *it << " ";  // 10 20 30 40 50
```

## Види ітераторів

```c++
std::vector<int> v = {1, 2, 3, 4, 5};

v.begin();          // ітератор на початок
v.end();            // ітератор на "після кінця"
v.rbegin();         // реверсний початок
v.rend();           // реверсний кінець

// Const версії:
v.cbegin();         // const_iterator
v.cend();
```

## Операції з ітераторами

```c++
std::vector<int> v = {10, 20, 30};

auto it = v.begin();

*it;        // 10 — розіменування
it[2];      // 30 — доступ за індексом
++it;       // перехід до наступного
--it;       // перехід до попереднього
it += 2;    // зсув на 2
auto dist = it - v.begin();  // різниця (кількість елементів)
```

## Зміна через ітератор

```c++
std::vector<int> v = {1, 2, 3, 4, 5};

for (auto it = v.begin(); it != v.end(); ++it)
    *it *= 2;  // {2, 4, 6, 8, 10}
```

## Ітератори та алгоритми

```c++
#include <algorithm>
#include <vector>

std::vector<int> v = {5, 2, 8, 1, 9};

std::sort(v.begin(), v.end());         // сортування
auto it = std::find(v.begin(), v.end(), 8);  // пошук
std::reverse(v.begin(), v.end());      // реверс
```

## Приклад: пошук

```c++
#include <iostream>
#include <vector>
#include <algorithm>

int main()
{
    std::vector<int> v = {4, 7, 1, 9, 3, 8};
    
    auto it = std::find(v.begin(), v.end(), 9);
    
    if (it != v.end())
    {
        std::cout << "Знайдено: " << *it << "\n";
        std::cout << "Позиція: " << (it - v.begin()) << "\n";
    }
    else
        std::cout << "Не знайдено\n";
    
    return 0;
}
```

## Коли використовувати?

| Контейнер | Індекс | Ітератор |
|-----------|--------|----------|
| vector | Так (O(1)) | Так |
| list | Ні | Так |
| map | Ні | Так |
| set | Ні | Так |

## Правило

Ітератори — універсальний спосіб обходу будь-яких контейнерів. Для простого обходу vector/array можна використовувати індекси.
