# Урок №101. Алгоритми в Стандартній бібліотеці С++

Стандартна бібліотека C++ надає багато готових алгоритмів у `<algorithm>`.

## Підключення

```c++
#include <algorithm>
#include <numeric>  // для numeric алгоритмів
```

## Огляд

Більшість алгоритмів працюють з діапазонами [begin, end).

## Основні алгоритми

### Пошук

```c++
std::vector<int> v = {1, 3, 5, 7, 9, 5};

auto it = std::find(v.begin(), v.end(), 5);      // перше входження 5
auto it2 = std::find_if(v.begin(), v.end(), 
    [](int x) { return x > 5; });                 // перший > 5
bool has = std::binary_search(v.begin(), v.end(), 5);  // якщо відсортовано
int count = std::count(v.begin(), v.end(), 5);   // кількість 5
```

### Сортування

```c++
std::vector<int> v = {5, 2, 8, 1, 9};

std::sort(v.begin(), v.end());               // {1, 2, 5, 8, 9}
std::stable_sort(v.begin(), v.end());        // стабільне сортування
std::partial_sort(v.begin(), v.begin()+3, v.end());  // часткове

// Реверс
std::reverse(v.begin(), v.end());            // {9, 8, 5, 2, 1}

// Перемішування
std::shuffle(v.begin(), v.end(), std::mt19937{});
```

### Зміна

```c++
std::vector<int> v = {1, 2, 3, 4, 5};

std::for_each(v.begin(), v.end(), [](int& x) { x *= 2; });  // {2, 4, 6, 8, 10}

std::transform(v.begin(), v.end(), v.begin(), 
    [](int x) { return x * x; });  // квадрати

std::fill(v.begin(), v.end(), 0);       // заповнити 0
std::replace(v.begin(), v.end(), 3, 33); // замінити 3→33
```

### Копіювання та видалення

```c++
std::vector<int> src = {1, 2, 3, 4, 5};
std::vector<int> dst(5);

std::copy(src.begin(), src.end(), dst.begin());  // копіювання

// Видалення (remove-erase idiom)
auto new_end = std::remove(v.begin(), v.end(), 3);  // переміщує 3 в кінець
v.erase(new_end, v.end());  // фізично видаляє
```

### Порівняння

```c++
std::vector<int> a = {1, 2, 3};
std::vector<int> b = {1, 2, 3};

bool eq = std::equal(a.begin(), a.end(), b.begin());  // true
auto pair = std::mismatch(a.begin(), a.end(), b.begin());  // перша відмінність
```

### Мінімум/Максимум

```c++
int x = std::min(5, 10);        // 5
int y = std::max(5, 10);        // 10
auto [min, max] = std::minmax({5, 2, 8, 1, 9});  // {1, 9}

auto it = std::min_element(v.begin(), v.end());   // ітератор на мінімум
auto it2 = std::max_element(v.begin(), v.end());  // ітератор на максимум
```

### Числові алгоритми (<numeric>)

```c++
#include <numeric>

std::vector<int> v = {1, 2, 3, 4, 5};

int sum = std::accumulate(v.begin(), v.end(), 0);  // 15
int prod = std::accumulate(v.begin(), v.end(), 1, std::multiplies<>());  // 120

std::vector<int> diff(v.size());
std::adjacent_difference(v.begin(), v.end(), diff.begin());  // {1, 1, 1, 1, 1}
```

## Приклад

```c++
#include <iostream>
#include <vector>
#include <algorithm>

int main()
{
    std::vector<int> v = {5, 2, 8, 1, 9, 3, 7, 4, 6};
    
    std::sort(v.begin(), v.end());
    
    auto it = std::find(v.begin(), v.end(), 7);
    if (it != v.end())
        std::cout << "Знайдено: " << *it << "\n";
    
    std::for_each(v.begin(), v.end(), [](int x) {
        std::cout << x << " ";
    });
    
    return 0;
}
```

## Правило

Не пишіть алгоритми самі, якщо вони вже є в стандартній бібліотеці. Вивчіть основні алгоритми — вони заощадять багато часу.
