# Урок №83. Клас std::string_view

`std::string_view` (C++17) — це неволодіюче представлення рядка. Він "дивиться" на рядок, не копіюючи його.

## Навіщо?

При передачі рядків у функції часто створюються зайві копії:

```c++
void process(std::string s);  // копіює! (навіть якщо тільки читає)
void process(const std::string& s);  // краще, але не працює з const char*
process("Hello");  // створюється тимчасовий std::string
```

## Рішення: std::string_view

```c++
#include <string_view>

void process(std::string_view s)  // не копіює!
{
    for (char c : s)  // можна ітерувати
        std::cout << c;
    
    std::cout << s.substr(0, 3);  // теж view
}

// Працює з усім:
std::string str = "Hello";
process(str);                 // const std::string&
process("Hello");             // const char*
process("Hello"sv);           // літерал string_view
```

## Створення

```c++
#include <string>
#include <string_view>

std::string s = "Hello";
std::string_view sv1 = s;        // з std::string
std::string_view sv2 = "Hello";  // з літерала
std::string_view sv3 = s.c_str(); // з C-string

using namespace std::string_view_literals;
auto sv4 = "Hello"sv;  // std::string_view літерал
```

## Що НЕ МОЖНА робити

```c++
std::string_view sv = "Hello";

// auto s = sv;  // OK — копіює view, не рядок
// std::string s(sv);  // створює копію

// Не можна змінювати через view:
// sv[0] = 'h';  // ПОМИЛКА! (якщо рядок константний)

// Небезпека: висить посилання
std::string_view getDangling()
{
    std::string s = "temp";
    return s;  // НЕБЕЗПЕЧНО! s знищується, view "висить"
}
```

## Коли використовувати?

```c++
// Добре: параметр функції, яка тільки читає рядок
void print(std::string_view msg);

// Добре: константи
constexpr std::string_view greeting = "Hello, world!";

// Не підходить: якщо потрібна зміна рядка
void modify(std::string& s);  // потрібен std::string&

// Не підходить: якщо потрібно зберігати рядок
class Person
{
    std::string name;  // БЕЗ string_view — потрібне володіння
};
```

## Приклад

```c++
#include <iostream>
#include <string_view>

bool startsWith(std::string_view str, std::string_view prefix)
{
    return str.substr(0, prefix.size()) == prefix;
}

int main()
{
    std::cout << startsWith("Hello world", "Hello");  // true
    std::cout << startsWith("Hi world", "Hello");     // false
    return 0;
}
```

## Правило

Використовуйте `std::string_view` для параметрів функцій, які тільки читають рядок. Ніколи не зберігайте `string_view` як член класу (якщо не впевнені, що рядок "живе" довше).
