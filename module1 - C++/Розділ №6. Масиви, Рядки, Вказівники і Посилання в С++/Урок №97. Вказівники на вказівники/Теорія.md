# Урок №97. Вказівники на вказівники

Вказівник може вказувати на інший вказівник.

## Синтаксис

```c++
int x = 5;
int* ptr = &x;     // вказівник на int
int** pptr = &ptr; // вказівник НА вказівник на int
```

## Доступ до значення

```c++
int x = 5;
int* ptr = &x;
int** pptr = &ptr;

std::cout << pptr;     // адреса ptr
std::cout << *pptr;    // адреса x (значення ptr)
std::cout << **pptr;   // 5 (значення x)
```

## Візуалізація

```
pptr → ptr → x
       0x100  0x200  5
```

## Динамічний двовимірний масив

```c++
int rows = 3, cols = 4;

int** matrix = new int*[rows];  // масив вказівників

for (int i = 0; i < rows; ++i)
    matrix[i] = new int[cols];  // кожен рядок

// Використання:
matrix[1][2] = 42;

// Звільнення:
for (int i = 0; i < rows; ++i)
    delete[] matrix[i];
delete[] matrix;
```

## Вказівник на вказівник у функціях

```c++
void allocate(int** ptr, int size)
{
    *ptr = new int[size];  // змінюємо, на що вказує ptr
    for (int i = 0; i < size; ++i)
        (*ptr)[i] = i * 10;
}

int main()
{
    int* arr = nullptr;
    allocate(&arr, 5);
    
    for (int i = 0; i < 5; ++i)
        std::cout << arr[i] << " ";  // 0 10 20 30 40
    
    delete[] arr;
    return 0;
}
```

## Три рівні (**)

Трапляється рідко, але можливо:

```c++
int x = 5;
int* ptr = &x;
int** pptr = &ptr;
int*** ppptr = &pptr;

***ppptr;  // 5
```

## Приклад: масив рядків

```c++
const char** names = new const char*[3];
names[0] = "Анна";
names[1] = "Іван";
names[2] = "Марія";

for (int i = 0; i < 3; ++i)
    std::cout << names[i] << "\n";

delete[] names;
```

## Правило

Вказівники на вказівники використовуються рідко. Основне використання — двовимірні динамічні масиви та зміна вказівника у функції.
