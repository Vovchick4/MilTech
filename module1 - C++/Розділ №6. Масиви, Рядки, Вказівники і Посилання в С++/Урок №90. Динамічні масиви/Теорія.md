# Урок №90. Динамічні масиви

Динамічний масив — масив, розмір якого визначається під час виконання.

## new[] та delete[]

```c++
int n = 10;
int* arr = new int[n];  // виділення

// Використання як звичайного масиву:
arr[0] = 5;
arr[1] = 10;

delete[] arr;  // звільнення
```

## Отримання розміру від користувача

```c++
#include <iostream>

int main()
{
    int size;
    std::cout << "Розмір масиву: ";
    std::cin >> size;
    
    int* arr = new int[size];
    
    for (int i = 0; i < size; ++i)
    {
        std::cout << "arr[" << i << "] = ";
        std::cin >> arr[i];
    }
    
    int sum = 0;
    for (int i = 0; i < size; ++i)
        sum += arr[i];
    
    std::cout << "Сума: " << sum << "\n";
    std::cout << "Середнє: " << static_cast<double>(sum) / size << "\n";
    
    delete[] arr;
    return 0;
}
```

## Функція з динамічним масивом

```c++
int* createArray(int size)
{
    int* arr = new int[size];
    for (int i = 0; i < size; ++i)
        arr[i] = i * 10;
    return arr;  // повертаємо вказівник
}

int main()
{
    int* data = createArray(5);
    
    for (int i = 0; i < 5; ++i)
        std::cout << data[i] << " ";
    
    delete[] data;  // не забуваємо!
    return 0;
}
```

## Зміна розміру (ручне розширення)

```c++
int* arr = new int[3] {1, 2, 3};

// Потрібен більший масив:
int* bigger = new int[5];
for (int i = 0; i < 3; ++i)
    bigger[i] = arr[i];
bigger[3] = 4;
bigger[4] = 5;

delete[] arr;   // звільняємо старий
arr = bigger;   // переприсвоюємо
```

## Проблеми

```c++
// 1. Забули delete[] — виточка
int* arr = new int[100];
// delete[] arr;  // забули!

// 2. Неправильний delete
int* arr = new int[10];
// delete arr;  // НЕПРАВИЛЬНО! потрібно delete[]

// 3. Використання після delete
int* arr = new int[5];
delete[] arr;
// arr[0] = 10;  // НЕБЕЗПЕЧНО!
```

## Сучасний підхід: std::vector

```c++
#include <vector>

std::vector<int> vec(10);  // 10 елементів, автоматичне керування
vec.push_back(11);         // автоматичне розширення
// delete не потрібен!
```

## Правило

Для динамічних масивів у 99% випадків використовуйте `std::vector`, а не `new[]`. Ручне керування пам'яттю — тільки при роботі з низькорівневим кодом.
