# Урок №94. Оператор доступу до членів через вказівник

Для доступу до членів структури/класу через вказівник використовується `->`.

## Проблема

```c++
struct Point { int x, y; };

Point p { 10, 20 };
Point* ptr = &p;

// Через вказівник:
(*ptr).x = 30;  // потрібні дужки — незручно
```

## Рішення: оператор ->

```c++
ptr->x = 30;  // еквівалентно (*ptr).x
ptr->y = 40;
```

## Приклад

```c++
#include <iostream>

struct Student
{
    std::string name;
    int age;
    double grade;
};

int main()
{
    Student s { "Олексій", 20, 95.5 };
    Student* ptr = &s;
    
    // Доступ через вказівник
    ptr->age = 21;
    std::cout << ptr->name << " " << ptr->age;
    
    return 0;
}
```

## Вказівник на динамічний об'єкт

```c++
struct Point { int x, y; };

Point* p = new Point { 5, 10 };
p->x = 15;
p->y = 25;
delete p;
```

## Вказівник на масив структур

```c++
struct Book { std::string title; int year; };

Book* lib = new Book[3] {
    {"C++ Basics", 2020},
    {"Algorithms", 2021},
    {"Design Patterns", 2022}
};

// Доступ:
lib[0].title;       // через індекс
(lib + 1)->year;    // через адресну арифметику

delete[] lib;
```

## Функція з вказівником на структуру

```c++
void printBook(const Book* book)
{
    std::cout << book->title << " (" << book->year << ")\n";
}
```

## Для посилань використовується .

Посилання — псевдонім, тому використовується `.`:

```c++
void printBook(const Book& book)
{
    std::cout << book.title << " (" << book.year << ")\n";
}
```

## Правило

Використовуйте `->` для доступу до членів через вказівник. Використовуйте `.` для доступу через об'єкт або посилання.

## Тест

2. Чим `ptr->x` відрізняється від `(*ptr).x`?
3. Який оператор використовується для доступу через посилання?
