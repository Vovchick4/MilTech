# Урок №77. Масиви

`Масив` (array) — це послідовний набір елементів одного типу, розташованих у пам'яті один за одним.

## Оголошення

```c++
int arr[5];               // масив з 5 int-ів (неініціалізований)
int arr[5] = {1, 2, 3, 4, 5};  // ініціалізація
int arr[] = {1, 2, 3};    // розмір виводиться (3)
```

## Індексація

Елементи нумеруються з 0:

```c++
int arr[5] = {10, 20, 30, 40, 50};

std::cout << arr[0];  // 10 (перший елемент)
std::cout << arr[2];  // 30
std::cout << arr[4];  // 50 (останній)
arr[1] = 25;          // зміна другого елемента
```

## Типові помилки

```c++
int arr[3] = {1, 2, 3};
std::cout << arr[3];  // НЕБЕЗПЕЧНО! вихід за межі — невизначена поведінка

int arr2[3] = {1, 2, 3, 4};  // ПОМИЛКА! забагато елементів
```

## Масиви та функції

```c++
void printArray(int arr[], int size)
{
    for (int i = 0; i < size; ++i)
        std::cout << arr[i] << " ";
}

int main()
{
    int data[] = {5, 10, 15, 20};
    printArray(data, 4);  // 5 10 15 20
    return 0;
}
```

## Розмір масиву

```c++
int arr[] = {1, 2, 3, 4, 5};
int size = sizeof(arr) / sizeof(arr[0]);  // 5
// В C++17: std::size(arr)
```

## Приклад

```c++
#include <iostream>

int main()
{
    int scores[5] = {85, 90, 78, 92, 88};
    
    int sum = 0;
    for (int i = 0; i < 5; ++i)
        sum += scores[i];
    
    double avg = static_cast<double>(sum) / 5;
    std::cout << "Середнє: " << avg;
    return 0;
}
```
