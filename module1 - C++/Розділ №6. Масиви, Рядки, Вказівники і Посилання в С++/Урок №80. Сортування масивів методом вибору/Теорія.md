# Урок №80. Сортування масивів методом вибору

`Сортування вибором` (selection sort) — один з найпростіших алгоритмів сортування.

## Як це працює

1. Знайти найменший елемент у масиві
2. Поміняти його місцями з першим елементом
3. Знайти найменший елемент серед решти
4. Поміняти його місцями з другим елементом
5. Повторювати, поки масив не відсортовано

## Візуалізація

```
Початковий: [5, 3, 8, 1, 7]
Крок 1: [1, 3, 8, 5, 7]  (1 ↔ 5)
Крок 2: [1, 3, 8, 5, 7]  (3 на місці)
Крок 3: [1, 3, 5, 8, 7]  (5 ↔ 8)
Крок 4: [1, 3, 5, 7, 8]  (7 ↔ 8)
Готово:  [1, 3, 5, 7, 8]
```

## Реалізація

```c++
#include <iostream>

void selectionSort(int arr[], int size)
{
    for (int i = 0; i < size - 1; ++i)
    {
        int minIndex = i;
        
        // Пошук найменшого елемента в невідсортованій частині
        for (int j = i + 1; j < size; ++j)
        {
            if (arr[j] < arr[minIndex])
                minIndex = j;
        }
        
        // Обмін
        if (minIndex != i)
        {
            int temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }
    }
}

int main()
{
    int arr[] = {5, 3, 8, 1, 7};
    int size = sizeof(arr) / sizeof(arr[0]);
    
    selectionSort(arr, size);
    
    for (int i = 0; i < size; ++i)
        std::cout << arr[i] << " ";
    // 1 3 5 7 8
    
    return 0;
}
```

## Аналіз

- **Час:** O(n²) — повільний для великих масивів
- **Пам'ять:** O(1) — сортує на місці
- **Простота:** Легко зрозуміти та реалізувати

## Сортування бульбашкою (bubble sort) для порівняння

```c++
void bubbleSort(int arr[], int size)
{
    for (int i = 0; i < size - 1; ++i)
    {
        for (int j = 0; j < size - i - 1; ++j)
        {
            if (arr[j] > arr[j + 1])
            {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}
```

## Сортування зі std::sort (найкраще)

```c++
#include <algorithm>

int arr[] = {5, 3, 8, 1, 7};
int size = sizeof(arr) / sizeof(arr[0]);

std::sort(arr, arr + size);  // O(n log n)
```

## Правило

Для навчання пишіть сортування самостійно. Для реального коду використовуйте `std::sort`.
