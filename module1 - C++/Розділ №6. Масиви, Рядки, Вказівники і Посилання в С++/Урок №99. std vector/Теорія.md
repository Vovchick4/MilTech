# Урок №99. std::vector (вектори)

`std::vector` — динамічний масив, який автоматично керує своєю пам'яттю.

## Підключення

```c++
#include <vector>
```

## Створення

```c++
std::vector<int> v1;              // порожній
std::vector<int> v2(5);           // 5 елементів (0)
std::vector<int> v3(5, 10);       // 5 елементів (10)
std::vector<int> v4 = {1, 2, 3};  // список ініціалізації
std::vector<int> v5(v4);          // копія
```

## Доступ

```c++
std::vector<int> v = {10, 20, 30, 40, 50};

v[0] = 15;          // без перевірки
v.at(1) = 25;       // з перевіркою (безпечно)
v.front();          // перший (10)
v.back();           // останній (50)
```

## Додавання та видалення

```c++
std::vector<int> v;

v.push_back(10);   // додати в кінець: {10}
v.push_back(20);   // {10, 20}
v.push_back(30);   // {10, 20, 30}
v.pop_back();      // видалити останній: {10, 20}

// C++11: emplace_back (ефективніше)
v.emplace_back(40);  // {10, 20, 40}
```

## Вставка та видалення (повільно!)

```c++
std::vector<int> v = {1, 2, 3, 4, 5};

v.insert(v.begin() + 2, 10);  // {1, 2, 10, 3, 4, 5}
v.erase(v.begin() + 1);       // {1, 10, 3, 4, 5}
```

## Розмір та ємність

```c++
std::vector<int> v = {1, 2, 3, 4, 5};

v.size();      // 5 — кількість елементів
v.capacity();  // ємність (зазвичай більше за size)
v.empty();     // false — чи порожній?
v.reserve(100); // зарезервувати місце
v.shrink_to_fit(); // зменшити capacity до size
```

## Ітерація

```c++
std::vector<int> v = {10, 20, 30};

// Index-based
for (size_t i = 0; i < v.size(); ++i)
    std::cout << v[i] << " ";

// Range-based for
for (int x : v)
    std::cout << x << " ";

// Range-based for (зміна)
for (int& x : v)
    x *= 2;
```

## Приклад

```c++
#include <iostream>
#include <vector>

int main()
{
    std::vector<int> numbers;
    
    std::cout << "Вводьте числа (0 для завершення):\n";
    while (true)
    {
        int n;
        std::cin >> n;
        if (n == 0) break;
        numbers.push_back(n);
    }
    
    int sum = 0;
    for (int x : numbers)
        sum += x;
    
    std::cout << "Сума: " << sum << "\n";
    std::cout << "Кількість: " << numbers.size() << "\n";
    std::cout << "Середнє: " << static_cast<double>(sum) / numbers.size() << "\n";
    
    return 0;
}
```

## std::vector — основний контейнер

| Операція | Складність |
|----------|-----------|
| Доступ за індексом | O(1) |
| Додавання в кінець | O(1) (амортизовано) |
| Вставка/видалення не в кінці | O(n) |
| Пошук | O(n) |

## Правило

`std::vector` — контейнер за замовчуванням у C++. Використовуйте його для динамічних масивів.
