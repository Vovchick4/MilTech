# Урок №59. Явна конвертація типів даних

`Явна конвертація` (casting) — перетворення типу, яке програміст вказує явно.

## C-style cast (застарілий)

```c++
int x = 10;
double y = (double)x / 3;   // C-style: 3.333
double z = double(x) / 3;   // те ж саме
```

**Недолік:** C-style cast може виконати БУДЬ-ЯКЕ перетворення — небезпечно.

## static_cast (C++, рекомендований)

```c++
int x = 10;
double y = static_cast<double>(x) / 3;  // 3.333
```

### Приклади

```c++
// Ціле ділення → дробове
int a = 7, b = 3;
double c = static_cast<double>(a) / b;  // 2.333

// char → int (ASCII код)
char ch = 'A';
int code = static_cast<int>(ch);  // 65

// int → char
int code = 66;
char ch = static_cast<char>(code);  // 'B'

// void* → конкретний тип
void* ptr = &x;
int* intPtr = static_cast<int*>(ptr);
```

## const_cast

Додає або забирає `const`:

```c++
void print(char* str) { std::cout << str; }

int main()
{
    const char* msg = "Hello";
    print(const_cast<char*>(msg));  // забираємо const
    return 0;
}
```

**Обережно:** зміна const-об'єкта через const_cast — невизначена поведінка!

## reinterpret_cast

Найнебезпечніший — перетворює будь-який вказівник на будь-який інший:

```c++
int x = 0x00414243;
char* ptr = reinterpret_cast<char*>(&x);
std::cout << *ptr;  // інтерпретуємо int як char (залежить від порядку байтів)
```

Використовується рідко — для низькорівневої роботи з пам'яттю.

## dynamic_cast

Для поліморфних типів (безпечне приведення в ієрархії класів):

```c++
class Base { virtual ~Base() {} };
class Derived : public Base { void foo() {} };

Base* b = new Derived;
Derived* d = dynamic_cast<Derived*>(b);
if (d) d->foo();  // перевірка перед використанням
```

## Порівняння cast-ів

| Cast | Коли використовувати |
|------|---------------------|
| `static_cast` | Стандартні перетворення (int→double, void*→тип) |
| `const_cast` | Додати/забрати const |
| `reinterpret_cast` | Низькорівневі перетворення вказівників |
| `dynamic_cast` | Безпечне приведення в ієрархії класів |

## Правило

Завжди використовуйте C++ cast-и (`static_cast`, `const_cast`, etc.) замість C-style `(int)`. Вони безпечніші та зрозуміліші.

## Приклад

```c++
#include <iostream>

int main()
{
    int a = 7, b = 3;
    
    // Без каста — ціле ділення
    std::cout << a / b << "\n";  // 2
    
    // static_cast — дробове
    std::cout << static_cast<double>(a) / b << "\n";  // 2.333
    
    // Інший спосіб
    std::cout << a / static_cast<double>(b) << "\n";  // 2.333
    
    return 0;
}
```
