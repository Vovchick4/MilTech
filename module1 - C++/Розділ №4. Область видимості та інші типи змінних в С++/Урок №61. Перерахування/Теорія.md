# Урок №61. Перерахування (перелічуваний тип даних)

`enum` (enumeration) — це тип даних, який містить набір іменованих цілочисельних констант.

## Оголошення

```c++
enum Color
{
    RED,     // 0
    GREEN,   // 1
    BLUE     // 2
};

enum Day
{
    MONDAY,     // 0
    TUESDAY,    // 1
    WEDNESDAY,  // 2
    THURSDAY,   // 3
    FRIDAY,     // 4
    SATURDAY,   // 5
    SUNDAY      // 6
};
```

## Використання

```c++
Color color { RED };

if (color == GREEN)
{
    std::cout << "Зелений\n";
}
```

## Перемикач (switch)

```c++
Day today { WEDNESDAY };

switch (today)
{
    case MONDAY:
        std::cout << "Початок тижня\n";
        break;
    case FRIDAY:
        std::cout << "Останній робочий\n";
        break;
    case SATURDAY:
    case SUNDAY:
        std::cout << "Вихідний!\n";
        break;
    default:
        std::cout << "Робочий день\n";
}
```

## Явне вказання значень

```c++
enum Animal
{
    CAT = -5,
    DOG,       // -4
    MOUSE,     // -3
    ELEPHANT = 10,
    LION,      // 11
    TIGER      // 12
};
```

## Виведення enum

```c++
Color c { BLUE };
std::cout << c;  // 2 (виводиться ціле число, не ім'я)

// Можна створити функцію для виводу імені:
std::string getColorName(Color c)
{
    switch (c)
    {
        case RED:   return "RED";
        case GREEN: return "GREEN";
        case BLUE:  return "BLUE";
        default:    return "UNKNOWN";
    }
}
```

## Недоліки звичайного enum

```c++
enum Color { RED, GREEN, BLUE };
enum Fruit { APPLE, BANANA, ORANGE };

Color c { RED };
Fruit f { APPLE };

if (c == f)  // КОМПІЛЮЄТЬСЯ! Порівнюються цілі числа!
{
    // Нелогічно, але працює
}

int x = RED;  // неявна конвертація в int — дозволено
```

## Правило

Використовуйте `enum` для групи пов'язаних констант (наприклад, дні тижня, кольори, стани). Для більш типобезпечних версій використовуйте `enum class` (наступний урок).

## Приклад

```c++
#include <iostream>

enum Direction
{
    NORTH,
    EAST,
    SOUTH,
    WEST
};

int main()
{
    Direction dir { EAST };
    
    if (dir == NORTH)
        std::cout << "Йдемо на північ\n";
    
    return 0;
}
```
