# Урок №52. Глобальні змінні

`Глобальна змінна` — це змінна, оголошена поза всіма функціями (у глобальній області видимості). Вона доступна з будь-якого місця програми.

## Оголошення

```c++
#include <iostream>

int g_x { 5 };       // глобальна змінна (префікс g_ — конвенція)

void foo()
{
    std::cout << g_x;  // доступ до глобальної змінної
}

int main()
{
    std::cout << g_x;  // також доступна тут
    foo();
    return 0;
}
```

## Область видимості та тривалість життя

- **Область видимості**: від точки оголошення до кінця файлу
- **Тривалість життя**: створюється до початку `main()`, знищується після завершення програми

```c++
int g_counter { 0 };  // існує весь час виконання програми

void increment() { ++g_counter; }

int main()
{
    increment();
    increment();
    std::cout << g_counter;  // 2
    return 0;
}
```

## Глобальна видимість у кількох файлах

Щоб зробити глобальну змінну доступною в інших файлах, використовуйте `extern`:

**globals.cpp**:
```c++
int g_value { 100 };  // визначення
```

**main.cpp**:
```c++
extern int g_value;   // оголошення (не визначення!)

int main()
{
    std::cout << g_value;  // 100
    return 0;
}
```

## Глобальні константи

Глобальні `const` змінні безпечніші — вони не можуть бути змінені:

```c++
// constants.h
#pragma once
namespace Constants
{
    constexpr double PI { 3.14159 };
    constexpr int MAX_STUDENTS { 30 };
    constexpr int DAYS_IN_WEEK { 7 };
}

// main.cpp
#include "constants.h"
std::cout << Constants::PI;
```

## Приклад використання

```c++
#include <iostream>

int g_mode { 1 };  // глобальний режим роботи

void printMode()
{
    if (g_mode == 1)
        std::cout << "Режим: редагування\n";
    else if (g_mode == 2)
        std::cout << "Режим: перегляд\n";
}

int main()
{
    printMode();   // Режим: редагування
    g_mode = 2;
    printMode();   // Режим: перегляд
    return 0;
}
```

## Глобальні змінні та статичність

```c++
static int g_fileStatic { 5 };  // видима тільки в цьому файлі
// (застарілий спосіб — краще використовувати анонімний namespace)
```

```c++
namespace            // анонімний namespace — файлова видимість
{
    int g_fileVar { 5 };
}
```

## Правило

Якщо глобальна змінна необхідна:
1. Використовуйте префікс `g_` для позначення
2. Робіть її `const` або `constexpr`, якщо можливо
3. Обмежуйте її файлову видимість (анонімний namespace або static)
