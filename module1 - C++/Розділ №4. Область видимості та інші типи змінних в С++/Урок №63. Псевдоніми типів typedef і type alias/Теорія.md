# Урок №63. Псевдоніми типів: typedef і type alias

## Псевдоніми типів

Псевдонім (alias) — це друге ім'я для типу. Оголосивши псевдонім, ви можете використовувати його замість оригінального типу.

## typedef (C-style, застарілий)

```c++
typedef long long int64_t;
typedef unsigned int uint;
typedef std::vector<int> IntVector;

int64_t x { 100 };        // long long
uint y { 50 };            // unsigned int
IntVector vec;             // std::vector<int>
```

## using (C++11, рекомендований)

```c++
using int64_t = long long;
using uint = unsigned int;
using IntVector = std::vector<int>;
```

## Чому `using` кращий?

1. **Читабельність** — природніший порядок слів:

```c++
typedef void (*FuncPtr)(int, double);  // складно
using FuncPtr = void(*)(int, double);  // зрозуміло
```

2. **Працює з шаблонами** (alias templates):

```c++
template<typename T>
using Vector = std::vector<T>;

Vector<int>    vec1;  // std::vector<int>
Vector<double> vec2;  // std::vector<double>

// З typedef це неможливо:
// template<typename T>
// typedef std::vector<T> Vector;  // ПОМИЛКА!
```

## Приклади використання

### Спрощення складних типів

```c++
// Без alias:
std::map<std::string, std::vector<int>>::iterator it;

// З alias:
using StudentGrades = std::map<std::string, std::vector<int>>;
StudentGrades::iterator it;
```

### Код, що легше змінювати

```c++
// Якщо потрібно змінити тип лічильника — змінюємо в одному місці
using Counter = int;
Counter students { 30 };
Counter teachers { 5 };
```

### Покращення читабельності

```c++
// Що означає std::map<std::string, std::string>?
// Краще:
using PhoneBook = std::map<std::string, std::string>;

// void (*)(int) — що це?
using EventHandler = void(*)(int);
```

## Коли використовувати?

```c++
// Для складних типів
using StringList = std::vector<std::string>;

// Для вказівників на функції
using Callback = void(*)(int, const std::string&);

// Для шаблонів
template<typename T>
using Matrix = std::vector<std::vector<T>>;

// Для скорочення довгих імен
using IntMap = std::map<int, int>;
```

## Приклад програми

```c++
#include <iostream>
#include <vector>
#include <string>

using StringVec = std::vector<std::string>;

void printStrings(const StringVec& vec)
{
    for (const auto& s : vec)
        std::cout << s << " ";
}

int main()
{
    StringVec names { "Олексій", "Марія", "Іван" };
    printStrings(names);
    return 0;
}
```

## Правило

Використовуйте `using` (type alias) замість `typedef`. Давайте псевдонімам змістовні імена, які пояснюють призначення типу в контексті.
