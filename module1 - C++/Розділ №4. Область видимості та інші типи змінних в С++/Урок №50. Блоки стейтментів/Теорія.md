# Урок №50. Блоки стейтментів

## Що таке блок?

`Блок` (або складений стейтмент) — це група стейтментів, об'єднаних фігурними дужками `{}`. Блок створює нову область видимості.

```c++
{
    int x = 5;
    std::cout << x;
}
```

## Блоки в різних конструкціях

```c++
int main()
{                     // початок блоку main
    if (true)
    {                 // початок блоку if
        std::cout << "Умова виконана\n";
    }                 // кінець блоку if
    
    for (int i = 0; i < 3; ++i)
    {                 // початок блоку for
        std::cout << i << "\n";
    }                 // кінець блоку for
    
    return 0;
}                     // кінець блоку main
```

## Блок без конструкції

Блок можна використовувати самостійно для обмеження області видимості:

```c++
int main()
{
    std::cout << "Початок\n";
    
    {
        int x = 5;
        std::cout << "Всередині блоку: " << x << "\n";
    }
    
    // std::cout << x;  // ПОМИЛКА! x не існує тут
    
    std::cout << "Кінець\n";
    return 0;
}
```

## Вкладені блоки

Блоки можуть бути вкладеними:

```c++
int main()
{
    int a = 1;
    
    {
        int b = 2;
        
        {
            int c = 3;
            std::cout << a << b << c;  // OK: видно всі
        }
        
        // std::cout << c;  // ПОМИЛКА! c не видно
        std::cout << a << b;  // OK
    }
    
    // std::cout << b;  // ПОМИЛКА! b не видно
    std::cout << a;  // OK
    
    return 0;
}
```

## Блоки визначають тривалість життя

Змінні всередині блоку створюються при вході в блок і знищуються при виході:

```c++
int main()
{
    {
        int temp = createLargeObject();
        // працюємо з temp
    }  // temp знищується, пам'ять звільняється
    
    // пам'ять вже вільна
    return 0;
}
```

## Блоки та умовні конструкції

```c++
if (x > 0)
    std::cout << "Додатне";   // один стейтмент — без дужок

if (x > 0)
{                               // кілька стейтментів — потрібні дужки
    std::cout << "Додатне\n";
    processPositive(x);
}
```

## Правило

Завжди використовуйте фігурні дужки для блоків if/else/for/while, навіть для одного стейтмента — це запобігає помилкам.
