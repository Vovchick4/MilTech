# Урок №54. Статичні змінні

`static` змінює тривалість життя та/або область видимості змінної.

## Статичні локальні змінні

Звичайна локальна змінна створюється при кожному виклику функції:

```c++
void counter()
{
    int count { 0 };
    ++count;
    std::cout << count << " ";  // 1 1 1 (завжди 1)
}
```

Статична локальна змінна створюється ОДИН раз і зберігає значення між викликами:

```c++
void counter()
{
    static int count { 0 };  // ініціалізується ТІЛЬКИ при першому виклику
    ++count;
    std::cout << count << " ";  // 1 2 3 4 ...
}

int main()
{
    counter();  // 1
    counter();  // 2
    counter();  // 3
    return 0;
}
```

## Як це працює?

Статичні локальні змінні:
- Ініціалізуються при ПЕРШОМУ виклику функції
- Зберігають значення між викликами
- Знищуються при завершенні програми
- Видимі тільки всередині функції

## Приклад: унікальний ID

```c++
int generateID()
{
    static int s_id { 0 };  // s_ — префікс для static
    return ++s_id;
}

int main()
{
    std::cout << generateID() << "\n";  // 1
    std::cout << generateID() << "\n";  // 2
    std::cout << generateID() << "\n";  // 3
    return 0;
}
```

## Статичні глобальні змінні (застаріло)

Раніше `static` використовувався для обмеження видимості глобальної змінної файлом:

```c++
// file.cpp
static int g_fileOnly { 5 };  // доступна тільки в цьому файлі
```

Сучасний спосіб — анонімний namespace:

```c++
namespace
{
    int g_fileOnly { 5 };  // файлова видимість
}
```

## static у класах (попередній перегляд)

```c++
class Counter
{
    static int s_total;  // оголошення статичного члена
public:
    Counter() { ++s_total; }
};

int Counter::s_total { 0 };  // визначення
```

## Коли використовувати static локальні змінні?

1. **Лічильники** — ID генератори, кількість викликів
2. **Одноразова ініціалізація** — ресурси, що ініціалізуються один раз
3. **Стан** — збереження стану між викликами

## Правило

Використовуйте `static` для локальних змінних, коли потрібно зберегти значення між викликами функції. Пам'ятайте про потокову безпеку в багатопотокових програмах.
