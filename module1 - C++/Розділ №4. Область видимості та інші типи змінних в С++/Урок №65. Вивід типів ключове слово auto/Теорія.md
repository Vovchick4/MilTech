# Урок №65. Вивід типів: ключове слово auto

`auto` — це ключове слово, яке дозволяє компілятору автоматично визначити тип змінної на основі її ініціалізатора.

## Базове використання

```c++
auto x = 5;            // int
auto y = 3.14;         // double
auto c = 'A';          // char
auto b = true;         // bool
auto s = "Hello";       // const char* (не std::string!)
auto str = std::string("Hello");  // std::string
```

## auto зі складними типами

Найбільша користь auto — спрощення складних типів:

```c++
// Без auto (складно читати):
std::vector<std::string> names;
std::vector<std::string>::iterator it = names.begin();

// З auto:
auto it = names.begin();  // тип виводиться автоматично

// Ще приклади:
std::map<std::string, int> scores;
auto iter = scores.find("Alex");  // складний тип — автоматично

auto result = someComplexFunction();  // не потрібно знати тип
```

## auto та const

```c++
const int x = 5;
auto y = x;         // y — int (const відкидається)
const auto z = x;   // z — const int

const int& ref = x;
auto r = ref;       // r — int (посилання відкидається)
auto& r2 = ref;     // r2 — const int&
```

## auto з range-based for

```c++
std::vector<int> vec { 1, 2, 3, 4, 5 };

// Копіювання (якщо потрібно змінювати — не підходить)
for (auto x : vec) { std::cout << x; }

// За посиланням (ефективніше, можна змінювати)
for (auto& x : vec) { x *= 2; }

// Const посилання (тільки читання)
for (const auto& x : vec) { std::cout << x; }
```

## auto у функціях (C++14)

C++14 дозволяє auto для типу повернення:

```c++
auto add(int a, int b)
{
    return a + b;  // auto → int
}

// C++20: auto для параметрів (шаблонні функції)
auto max(auto a, auto b)
{
    return (a > b) ? a : b;
}
```

## auto та decltype

```c++
int x = 5;
decltype(x) y = 10;  // y — int (тип такий самий як x)

// auto — виводить тип з ІНІЦІАЛІЗАТОРА
// decltype — виводить тип з ВИРАЗУ
```

## Коли використовувати auto?

```c++
// Добре: складні типи
auto it = container.begin();
auto ptr = std::make_unique<MyClass>();

// Добре: range-based for
for (const auto& item : container)

// Добре: лямбди
auto func = [](int x) { return x * 2; };

// Не дуже: проста змінна (краще явно)
auto x = 5;     // int — очевидно, але краще написати int
int y = 5;      // зрозуміліше
```

## Правило

Використовуйте `auto`, коли:
- Тип складний або довгий
- Тип очевидний з контексту
- Тип залежить від шаблону

Уникайте `auto` там, де тип неочевидний:

```c++
auto result = doSomething();  // що повертає doSomething()?
```
