# Урок №60. Введення в std::string

`std::string` — це клас зі Стандартної бібліотеки C++ для роботи з рядками.

## Підключення

```c++
#include <string>  // не забувайте!
```

## Створення та ініціалізація

```c++
#include <string>

std::string s1;               // порожній рядок
std::string s2 { "Hello" };   // з літерала
std::string s3 { "Hello", 3 }; // "Hel" (перші 3 символи)
std::string s4 { 5, 'A' };    // "AAAAA" (5 символів A)
std::string s5 { s2 };        // копія: "Hello"
```

## Виведення

```c++
std::string name { "Alex" };
std::cout << name;           // Alex
std::cout << name[0];        // A (перший символ)
```

## Введення

```c++
std::string name;
std::cin >> name;   // читає до першого пробілу
// Ввід: "John Doe"
// name = "John"

// Читання всього рядка (з пробілами)
std::string fullName;
std::getline(std::cin, fullName);  // читає до символу нового рядка
```

## Основні операції

```c++
std::string s1 { "Hello" };
std::string s2 { "World" };

// Конкатенація (+)
std::string s3 = s1 + " " + s2;  // "Hello World"
s1 += "!!!";  // s1 = "Hello!!!"

// Довжина
int len = s1.length();   // 8
int size = s1.size();    // 8 (те саме)

// Доступ за індексом
char first = s1[0];      // 'H'
char last = s1[s1.length() - 1];  // '!'

// Порівняння
if (s1 == s2) { }
if (s1 != s2) { }
if (s1 < s2) { }  // лексикографічне
```

## Корисні методи

```c++
std::string s { "Hello, World!" };

// Пошук
int pos = s.find("World");   // 7 (позиція або string::npos)

// Підрядок
std::string sub = s.substr(7, 5);  // "World" (позиція, довжина)

// Вставка
s.insert(7, "Dear ");  // "Hello, Dear World!"

// Видалення
s.erase(7, 5);         // видаляє "Dear "

// Заміна
s.replace(7, 5, "Everyone");  // замінює "World" на "Everyone"

// Перевірка
if (s.empty()) { }            // чи порожній?
if (s.find("Hello") != std::string::npos) { }  // чи містить підрядок?
```

## Рядки C++ vs C-style

```c++
// C-style рядок (масив символів)
const char* cstr = "Hello";

// C++ рядок (std::string)
std::string cppstr = "Hello";

// Конвертація
const char* back = cppstr.c_str();  // std::string → C-string
```

## Приклад програми

```c++
#include <iostream>
#include <string>

int main()
{
    std::cout << "Введіть ім'я: ";
    std::string name;
    std::getline(std::cin, name);
    
    std::cout << "Введіть вік: ";
    int age;
    std::cin >> age;
    
    std::string message = name + " має " + std::to_string(age) + " років";
    std::cout << message << "\n";
    std::cout << "Довжина імені: " << name.length() << "\n";
    
    return 0;
}
```

## Правило

Використовуйте `std::string` замість C-style рядків. Він безпечніший, зручніший і має багато корисних методів.
