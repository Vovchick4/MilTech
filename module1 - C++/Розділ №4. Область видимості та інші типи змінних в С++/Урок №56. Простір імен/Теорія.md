# Урок №56. Простір імен

`Простір імен` (namespace) — це механізм групування ідентифікаторів для уникнення конфліктів імен.

## Проблема

```c++
// library1.h
void print();  // друк у форматі library1

// library2.h
void print();  // друк у форматі library2

// main.cpp
#include "library1.h"
#include "library2.h"
print();  // який print() викликати? КОНФЛІКТ!
```

## Рішення: namespace

```c++
// library1.h
namespace Lib1
{
    void print();  // Lib1::print()
}

// library2.h
namespace Lib2
{
    void print();  // Lib2::print()
}

// main.cpp
int main()
{
    Lib1::print();  // викликає Lib1::print
    Lib2::print();  // викликає Lib2::print
    return 0;
}
```

## Синтаксис

```c++
namespace MyName
{
    int x { 5 };
    void func() { std::cout << "Hello\n"; }
    
    namespace Inner  // вкладений namespace
    {
        int y { 10 };
    }
}

// Доступ:
MyName::x;           // 5
MyName::func();      // Hello
MyName::Inner::y;    // 10
```

## Визначення функцій поза namespace

```c++
// заголовковий файл
namespace Math
{
    int add(int a, int b);
    int multiply(int a, int b);
}

// файл реалізації
#include "math.h"

int Math::add(int a, int b)
{
    return a + b;
}

int Math::multiply(int a, int b)
{
    return a * b;
}
```

## Простір імен std

`std` — це простір імен Стандартної бібліотеки C++. Всі стандартні функції, об'єкти та типи знаходяться в ньому:

```c++
std::cout << "Hello";
std::string name = "Alex";
std::vector<int> data;
std::sort(data.begin(), data.end());
```

## Розширення namespace

Namespace можна доповнювати в різних файлах:

```c++
// file1.cpp
namespace MyLib
{
    void func1();
}

// file2.cpp
namespace MyLib
{
    void func2();  // додаємо до того ж namespace
}
```

## Анонімний namespace

Namespace без імені — обмежує видимість файлом (альтернатива `static`):

```c++
namespace
{
    int g_internal { 5 };       // файлова видимість
    void internalFunc() { }
}
```

## Правило

Ніколи не пишіть `using namespace std;` у заголовкових файлах — це "забруднює" глобальний простір імен для всіх, хто включає ваш файл.
