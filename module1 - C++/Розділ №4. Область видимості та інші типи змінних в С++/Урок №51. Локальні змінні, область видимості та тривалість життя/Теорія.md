# Урок №51. Локальні змінні, область видимості та тривалість життя

## Локальні змінні

`Локальна змінна` — це змінна, оголошена всередині функції або блоку `{}`.

```c++
void someFunction()
{
    int x { 5 };      // локальна змінна
    std::cout << x;
}
```

## Область видимості (scope)

`Область видимості` визначає, де змінна доступна за ім'ям:

```c++
int main()
{
    int x { 5 };  // область видимості: від оголошення до кінця блоку
    
    if (x > 0)
    {
        int y { 10 };  // область видимості: тільки в цьому if-блоці
        std::cout << x << y;  // OK: x і y доступні
    }
    
    // std::cout << y;  // ПОМИЛКА! y не видно
    std::cout << x;  // OK
}
```

## Тривалість життя (lifetime)

`Тривалість життя` — час, коли змінна існує в пам'яті:

```c++
void example()
{
    int x { 5 };    // x створюється (виділяється пам'ять)
    // працюємо з x
    x = x + 3;
    std::cout << x;
}                   // x знищується (пам'ять звільняється)
```

## Послідовність створення та знищення

Змінні створюються в порядку оголошення, знищуються в зворотному порядку (LIFO):

```c++
void func()
{
    int a { 1 };  // 1) створюється a
    int b { 2 };  // 2) створюється b
    {
        int c { 3 };  // 3) створюється c
    }                 // 4) знищується c
    int d { 4 };  // 5) створюється d
}                 // 6) знищується d, b, a (у зворотному порядку)
```

## Локальні змінні в різних функціях

Кожна функція має власний набір локальних змінних:

```c++
void foo()
{
    int x { 5 };
    std::cout << x;  // 5
}

void bar()
{
    int x { 10 };    // інша x! не пов'язана з x у foo()
    std::cout << x;  // 10
}

int main()
{
    foo();  // 5
    bar();  // 10
    return 0;
}
```

## Змінна в циклі

```c++
for (int i = 0; i < 3; ++i)
{
    // i доступна тільки в циклі
    int temp = i * 2;  // temp створюється на кожній ітерації
    std::cout << temp << " ";
}
// i не видна тут
```

## Правило

Оголошуйте змінні якомога ближче до місця їх першого використання і в найменшій області видимості, де вони потрібні.
