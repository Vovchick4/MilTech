# Урок №62. Класи enum

`enum class` (або scoped enumeration) — покращена версія enum, введена в C++11. Вона вирішує проблеми звичайного enum.

## Оголошення

```c++
enum class Color
{
    RED,
    GREEN,
    BLUE
};

enum class Fruit
{
    APPLE,
    BANANA,
    ORANGE
};
```

## Використання

Обов'язково потрібно вказувати ім'я enum:

```c++
Color c { Color::RED };   // ТРЕБА: Color::
// Color c { RED };       // ПОМИЛКА! RED не видно без префікса
```

## Типобезпека

```c++
Color c { Color::RED };
Fruit f { Fruit::APPLE };

// if (c == f)  // ПОМИЛКА КОМПІЛЯЦІЇ! різні типи
// int x = c;  // ПОМИЛКА! немає неявної конвертації в int
```

## Явне приведення до int

```c++
Color c { Color::GREEN };
int value = static_cast<int>(c);  // 1 (явно, безпечно)
Color c2 = static_cast<Color>(2); // Color::BLUE
```

## Вказання базового типу

```c++
enum class Color : std::uint8_t  // займає 1 байт
{
    RED,
    GREEN,
    BLUE
};

enum class State : int  // явно int (можна не вказувати)
{
    OK = 200,
    NOT_FOUND = 404,
    ERROR = 500
};
```

## Приклад з функціями

```c++
#include <iostream>

enum class Operation
{
    ADD,
    SUBTRACT,
    MULTIPLY
};

int calculate(int a, int b, Operation op)
{
    switch (op)
    {
        case Operation::ADD:      return a + b;
        case Operation::SUBTRACT: return a - b;
        case Operation::MULTIPLY: return a * b;
        default:                  return 0;
    }
}

int main()
{
    int result = calculate(10, 5, Operation::SUBTRACT);
    std::cout << result;  // 5
    return 0;
}
```

## enum vs enum class

| Особливість | enum | enum class |
|-------------|------|------------|
| Контроль області | Глобальна | Через `EnumName::` |
| Неявна конвертація в int | Так | Ні |
| Порівняння різних типів | Дозволено | Заборонено |
| Можна вказати базовий тип | Так | Так |

## Правило

Завжди використовуйте `enum class` замість звичайного `enum`. Він безпечніший і запобігає конфліктам імен.

## Тест

2. Чи можна порівнювати `Color::RED` та `Fruit::APPLE`?
3. Чому `enum class` кращий за звичайний `enum`?
